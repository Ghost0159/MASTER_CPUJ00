MB_SetSendVolatCallback

構文

#include <nitro/mb/mb_gameinfo.h>

void 
MB_SetSendVolatCallback( MBSendVolatCallbackFunc callback, u32 timming );

引数

callback 親機ゲーム情報中のMBGameInfoVolatileをビーコンで送信する際に呼ばれるコールバック関数を設定します。
timming コールバック関数が呼ばれるタイミングを指定します。
MB_SEND_VOLAT_CALLBACK_TIMMING_BEFORE MBGameInfoVolatileの格納されたビーコンが送信設定される直前にコールバックが呼ばれます。
MB_SEND_VOLAT_CALLBACK_TIMMING_AFTER MBGameInfoVolatileの格納されたビーコンが送信設定された直後にコールバックが呼ばれます。

返り値

なし。

説明

MB親機ゲーム情報のうち、流動的に変化するMBGameInfoVolatileをビーコンで送る際のコールバックを設定することができます。
ここには、MB_SetUserVolatDataで設定するユーザデータも含まれており、このコールバック中でMB_SetUserVolatDataをコールすることで、次に送信するデータを設定することができます。

/* コールバック関数 */
 typedef void (*MBSendVolatCallbackFunc)( u32 ggid );
/* コールバックタイミング */
enum
{
    MB_SEND_VOLAT_CALLBACK_TIMMING_BEFORE,
    MB_SEND_VOLAT_CALLBACK_TIMMING_AFTER
};
