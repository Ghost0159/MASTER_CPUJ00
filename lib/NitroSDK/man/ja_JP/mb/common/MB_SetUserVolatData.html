MB_SetUserVolatData

構文

#include <nitro/mb/mb_gameinfo.h>

void  
MB_SetUserVolatData( u32 ggid, const u8 * userData, u32 size );

引数

ggid ビーコンにデータを設定するプログラムのggid。MB_RegisterFileで設定したggidを識別子として使用します。
userData 設定するデータへのポインタ。
size 設定するデータサイズ。最大８バイトまで。

返り値

なし。

説明

MB親機ゲーム情報のうち、流動的に変化するMBGameInfoVolatileの中にユーザデータを設定します。
設定したデータは、MB_GetUserVolatDataまたは、MBFakeScanCallback中のMBGameInfoのパラメータから取得することが可能です。

参照

MB_SetSendVolatCallback, MBGameInfo
