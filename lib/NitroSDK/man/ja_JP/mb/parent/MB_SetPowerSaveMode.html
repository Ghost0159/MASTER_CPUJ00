MB_SetPowerSaveMode

構文

#include <nitro/mb.h>

void MB_SetPowerSaveMode(BOOL enable);

引数

enable 省電力モードを有効にするならば TRUE, 無効にするならば FALSE
この設定は、デフォルトでは有効となっています。

返り値

なし

説明

MBライブラリの内部で駆動される通信の省電力モードを指定します。
この設定は、デフォルトでは有効となっています。

この関数は電力消費を考慮しなくてよい場面でより安定に駆動するためのオプションであり、
電源につながれていることを保証された動作環境でない限り通常のゲームアプリケーションがこれを使用すべきではありません。

この関数は、MB_Init関数の呼び出し以降、MB_StartParent関数を呼び出すまでのタイミングでのみ呼び出すことが可能です。
