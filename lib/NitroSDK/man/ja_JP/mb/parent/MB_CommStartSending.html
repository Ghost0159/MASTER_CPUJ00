MB_CommStartSending

構文

#include <nitro/mb.h>

BOOL MB_CommStartSending( u16 child_aid );

引数

child_aid ダウンロード開始通知を送信する子機を指す aid

返り値

指定した子機が接続許可を受けていれば TRUE を、そうでないなら FALSE を返します。

説明

現在親機から接続許可を受けている特定の子機に対してダウンロード開始通知を送信します。
指定した子機が接続許可を受けていれば TRUE を返します。
親機通信状態がMB_COMM_PSTATE_REQ_WAIT_TO_SENDである時、子機がダウンロード開始通知を待っていることを示します。
