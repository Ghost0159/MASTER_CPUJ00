MB_TYPE_*

定義

#include <nitro/mb.h>

/* ブートタイプ */
#define	MB_TYPE_ILLEGAL		0	/* 不正な状態 */
#define	MB_TYPE_NORMAL		1	/* ROM から起動 */
#define	MB_TYPE_MULTIBOOT	2	/* マルチブート子機 */


説明

プログラム自身の起動状態を表す定数です。
各値の意味は以下の通りです。
値 意味
MB_TYPE_ILLEGAL なんらかの不正な状態にあり、起動状態を判断できない状態です。
MB_TYPE_NORMAL NITRO-CARDからの通常の起動状態です。
MB_TYPE_MULTIBOOT 任意の親機からのマルチブート起動状態です。
