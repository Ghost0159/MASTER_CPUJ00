MBCommPStateCallback

定義

#include <nitro/mb.h>

typedef void (*MBCommPStateCallback)(u16 child_aid, u32 status, void* arg);

説明

親機の状態が変更するごとに呼び出されるコールバック関数の形式を定義します。
引数の意味は以下の通りです。
引数 意味
child_aid コールバックが特定の1台の子機に対する通知である場合、その子機の aid を示します。
この値を指定してMB_CommGetChildUser関数を呼び出すことで、
特定の子機についての詳細を取得することができます。
status MBCommPStateで表されるいずれかの状態を示します。
arg 通知される状態に応じた変数が渡されます。
各状態ごとの詳細は、MBCommPStateの説明を参照ください。
