CTRDG_Read*

構文

#include <nitro/ctrdg.h>
BOOL CTRDG_Read8( const u8* address, u8* rdata );
BOOL CTRDG_Read16( const u16* address, u16* rdata );
BOOL CTRDG_Read32( const u32* address, u32* rdata );

引数

address 読みこみアドレス
rdata 読み込んだデータを格納するポインタ

返り値

読み込みに成功すればTRUEを返します。カートリッジが存在せず読めなかった場合は FALSE を返します。

説明

カートリッジからデータを読み込みます。

CTRDG_Read8() は 8 ビットデータを読み込みます。
CTRDG_Read16() は 16 ビットデータを読み込みます。
CTRDG_Read32() は 32 ビットデータを読み込みます。

アクセスサイクルの設定は行いません。事前に行っておいてください。

参照

CTRDG_Init, CTRDG_DmaCopy*, CTRDG_CpuCopy*, CTRDG_Write*
