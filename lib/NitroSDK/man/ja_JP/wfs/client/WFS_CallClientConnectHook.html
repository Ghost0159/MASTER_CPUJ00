WFS_CallClientConnectHook

構文

#include <nitro/wfs.h>

void WFS_CallClientConnectHook(WFSClientContext *context, const WFSPeerInfo *peer);

引数

context WFSClientContext構造体のポインタ
peer 接続した親機の情報が格納されたWFSPeerInfo構造体のポインタ

返り値

なし

説明

親機へ接続したことをライブラリへ通知します。
アプリケーションはワイヤレス通信で親機へ接続した時にこの関数を呼び出す必要があります。
