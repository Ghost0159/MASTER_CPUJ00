WFS_InitClient

構文

#include <nitro/wfs.h>

void WFS_InitClient(WFSClientContext *context, void *userdata, WFSEventCallback callback, MIAllocator *allocator);

引数

context 初期化するWFSClientContext構造体のポインタ
userdata コンテキストに関連付ける任意のユーザ定義値
callback WFSライブラリからのイベント通知を受けるコールバック関数
不要ならNULLを指定します。
allocator ライブラリが内部で使用するアロケータ

返り値

なし

説明

WFSライブラリのクライアントコンテキスト構造体を初期化します。
WFSClientContext構造体を使用する前に、必ず1回だけこの関数を呼び出しておく必要があります。
この関数はWFSClientContext構造体の内容を初期化するだけなのでOSやワイヤレスの状態には一切影響を及ぼしません。
