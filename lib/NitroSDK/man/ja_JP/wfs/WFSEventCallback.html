WFSEventCallback

定義

#include <nitro/wfs.h>

typedef void (*WFSEventCallback)(void *context, WFSEventType event, void *argument);

説明

WFSライブラリで使用されるイベントコールバック形式のプロトタイプ宣言です。
各引数の意味は以下の通りです。

要素

context WFSServerContext構造体またはWFSClientContext構造体のポインタ
event コールバックの通知種別を示すWFSEventType列挙型の値
argument コールバックの通知種別ごとに異なる情報を保持した引数
通知種別ごとの引数の内容は、WFSEventTypeの説明を参照してください。

参照

WFS_InitServer WFS_InitClient
