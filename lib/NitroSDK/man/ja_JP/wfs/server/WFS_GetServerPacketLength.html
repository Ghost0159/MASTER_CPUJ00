WFS_GetServerPacketLength

構文

#include <nitro/wfs.h>

int WFS_GetServerPacketLength(const WFSServerContext *context);

引数

context WFSServerContext構造体のポインタ

返り値

現在設定されているMP送信パケットサイズ上限値

説明

親機がWFS_CallServerPacketSendHook関数で生成するMP送信パケットサイズの上限を取得します。
子機からの読み込みリクエストに応答する際、このパケットサイズが大きいほど転送効率は向上します。

参照

WFS_InitServer WFS_SetServerPacketLength
