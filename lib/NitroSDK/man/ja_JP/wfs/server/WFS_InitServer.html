WFS_InitServer

構文

#include <nitro/wfs.h>

void WFS_InitServer(WFSServerContext *context, void *userdata, WFSEventCallback callback, MIAllocator *allocator, int packet);

引数

context 初期化するWFSServerContext構造体のポインタ
userdata コンテキストに関連付ける任意のユーザ定義値
callback WFSライブラリからのイベント通知を受けるコールバック関数
不要ならNULLを指定します。
allocator ライブラリが内部で使用するアロケータ
packet WFSのために親機が使用可能なMP送信パケットサイズ
この設定はWFS_SetServerPacketLength関数で変更することもできます。

返り値

なし

説明

WFSライブラリのサーバコンテキスト構造体を初期化します。
WFSServerContext構造体を使用する前に、必ず1回だけこの関数を呼び出しておく必要があります。
この関数はWFSServerContext構造体の内容を初期化するだけなのでOSやワイヤレスの状態には一切影響を及ぼしません。
