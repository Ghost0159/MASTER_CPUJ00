WFS_CallServerConnectHook

構文

#include <nitro/wfs.h>

void WFS_CallServerConnectHook(WFSServerContext *context, const WFSPeerInfo *peer);

引数

context WFSServerContext構造体のポインタ
peer 接続した子機の情報が格納されたWFSPeerInfo構造体のポインタ

返り値

なし

説明

新規子機が接続したことをライブラリへ通知します。
アプリケーションはワイヤレス通信で子機が接続するたびにこの関数を呼び出す必要があります。
