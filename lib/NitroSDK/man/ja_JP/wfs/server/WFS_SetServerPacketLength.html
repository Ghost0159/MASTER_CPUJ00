WFS_SetServerPacketLength

構文

#include <nitro/wfs.h>

void WFS_SetServerPacketLength(WFSServerContext *context, int length);

引数

context WFSServerContext構造体のポインタ
length WFSのために使用可能なMP送信パケットサイズ上限値

返り値

なし

説明

親機がWFS_CallServerPacketSendHook関数で生成するMP送信パケットサイズの上限を設定します。
子機からの読み込みリクエストに応答する際、このパケットサイズが大きいほど転送効率は向上します。
この関数はWFS_InitServer関数で指定したパケットサイズを動的に変更したい場合に使用します。

WFS_GetServerConnectedBitmap WFS_GetServerBusyBitmap
