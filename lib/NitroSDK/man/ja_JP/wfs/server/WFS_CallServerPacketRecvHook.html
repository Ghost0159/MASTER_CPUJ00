WFS_CallServerPacketRecvHook

構文

#include <nitro/wfs.h>

void WFS_CallServerPacketRecvHook(WFSServerContext *context, const WFSPacketBuffer *packet);

引数

context WFSServerContext構造体のポインタ
packet 受信パケットの情報が格納されたWFSPacketBuffer構造体のポインタ
各メンバには以下の内容を設定して与える必要があります。
buffer 受信したパケットデータのバッファを指定します。
length 受信したサイズを指定します。
bitmap パケット送信元を示すAIDビットマップを指定します。
アプリケーション側で特殊な管理を行わなければ常に(1 << aid)で構いません。

返り値

なし

説明

子機から受信したMP通信1回分のパケットデータをWFSライブラリの内部的なプロトコルに変換します。 MP通信で受信したパケットデータをこの関数で解析することにより、親子間でWFS状態が更新されます。
アプリケーションはこの関数呼び出しとMP通信を適当な頻度で繰り返す必要があります。
