STD_ConvertCharSjisToUnicode

構文

#include <nitro/std.h>

SDK_INLINE int STD_ConvertCharSjisToUnicode(u16 *dst, const char *src);

引数

dst 変換先の Unicode 文字列バッファ
src 変換元の ShiftJIS 文字列バッファ

返り値

変換に成功した場合は、変換した文字数(= 1)を返します。 変換に失敗した場合は、-1 を返します。

説明

ShiftJIS 形式の文字を Unicode 形式の文字に変換します。
内部でSTD_ConvertStringSjisToUnicodeを呼び出して1文字だけの変換を行っています。
