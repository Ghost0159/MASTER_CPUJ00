STD_CompareString

構文

#include <nitro/std.h>

int STD_CompareString( const char* str1, const char* str2 );
int STD_StrCmp( const char* str1, const char* str2 );

引数

str1 比較する文字列のポインタを指定します。
str2 比較する文字列のポインタを指定します。

返り値

比較した文字列が同一ならば 0 を返します。

説明

文字列 str1 と、文字列 str2 を比較します。C標準関数の strcmp() と同じ動作をします。

STD_StrCmp() は STD_CompareString() の別名として define されています。

参照

STD_CompareNString, STD_CompareLString
