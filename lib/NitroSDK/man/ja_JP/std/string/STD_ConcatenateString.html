STD_ConcatenateString

構文

#include <nitro/std.h>

char* STD_ConcatenateString( char* str1, const char* str2 );
char* STD_StrCat( char* str1, const char* str2 );

引数

str1 連結先の文字列のポインタを指定します。
str2 連結元の文字列のポインタを指定します。

返り値

連結した文字列のポインタを返します。

説明

文字列 str2 を、文字列 str1 の終りに連結します。C標準関数の strcat() と同じ動作をします。

STD_StrCat() は STD_ConcatenateString() の別名として define されています。
