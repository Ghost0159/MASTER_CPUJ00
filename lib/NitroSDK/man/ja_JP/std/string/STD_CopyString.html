STD_CopyString

構文

#include <nitro/std.h>

char* STD_CopyString( char* destp, const char* srcp );
char* STD_StrCpy( char* destp, const char* srcp );

引数

destp コピー先のポインタを指定します。
srcp コピー元のポインタを指定します。

返り値

コピー先の文字列のポインタを返します。

説明

文字列 srcp を、文字列 destp へコピーします。C標準関数の strcpy() と同じ動作をします。

STD_StrCpy() は STD_CopyString() の別名として define されています。
