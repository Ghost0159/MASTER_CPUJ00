STD_CompareNString

構文

#include <nitro/std.h>

int STD_CompareNString( const char* str1, const char* str2, int len );
int STD_StrNCmp( const char* str1, const char* str2, int len );

引数

str1 比較する文字列のポインタを指定します。
str2 比較する文字列のポインタを指定します。
len 比較する文字数を指定します。

返り値

比較した文字列が同一ならば 0 を返します。

説明

文字列 str1 と、文字列 str2 を最初の len 文字のみ比較します。C標準関数の strncmp() と同じ動作をします。

STD_StrNCmp() は STD_CompareNString() の別名として define されています。

参照

STD_CompareString, STD_CompareLString
