STD_TVSPrintf

構文

#include <nitro/std.h>

int STD_TVSPrintf( char *dst, const char *fmt, va_list vlist );

引数

dst 出力文字列の格納先バッファ
fmt 出力書式文字列(フォーマット文字列)
vlist 可変引数リスト

返り値

出力した文字数を返します。 この文字数に終端文字 '\0' は含まれません。

説明

軽量版の書式文字列関数です。

格納先バッファのサイズを考慮しないことを除いて、 STD_TVSNPrintf() と等価です。

この関数は weak シンボルでコンパイルされますので、ユーザが独自で関数を定義すれば、それで上書きされます。

参照

STD_TSPrintf STD_TSNPrintf STD_TVSNPrintf
