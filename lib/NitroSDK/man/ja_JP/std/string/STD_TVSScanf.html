STD_TVSScanf

構文

#include <nitro/std.h>

int STD_TVSScanf( const char *src, const char *fmt, va_list vlist );

引数

src 入力文字列
fmt 入力書式文字列(フォーマット文字列)
vlist 可変引数リスト

返り値

書式変換が成功し実際に代入された回数を返します。

1回も書式変換が成功せず src の終端に達した場合は -1 を返します。

説明

軽量版の書式文字列関数です。

以下の機能について、vsscanf関数と同じ動作です。

・整数変換 (d, i, o, u, x, X, p)
・文字および文字列 (c, s)
・走査位置取得 (n)
・文字クラス ([])
