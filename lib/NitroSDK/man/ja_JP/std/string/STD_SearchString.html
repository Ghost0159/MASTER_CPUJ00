STD_SearchString

構文

#include <nitro/std.h>

char* STD_SearchString( char* srcp, const char* str );
char* STD_StrStr( char* srcp, const char* str );

引数

srcp 探される文字列のポインタを指定します。
str 探す文字列のポインタを指定します。

返り値

文字列 srcp の中で文字列 str が最初に現れる位置へのポインタを返します。
それがない場合には NULL を返します。

説明

文字列 srcp から、文字列 str を探します。 C標準関数の strstr() と同じ動作をします。

STD_StrStr() は STD_SearchString() の別名として define されています。
