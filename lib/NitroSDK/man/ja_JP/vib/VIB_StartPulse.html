VIB_StartPulse

構文

#include <nitro/vib.h>
void VIB_StartPulse(const VIBPulseState* state);

引数

state パルス振動のステータス。

返り値

なし

説明

与えられたステータスでパルス振動を開始します。
もし以前のパルス振動が終了していない場合は、いったん終了してから始めます。
ステータスはライブラリ側でコピーしますので、メモリを確保しておく必要はありません。

振動を開始する前に、ステータスがハードウェアによる制限を守っているかどうかチェックが行われます。制限の詳細については、VIBPulseState 関数をご参照ください。
制限に反していた場合、DEBUG 又は RELEASE ビルドでは、OS_Panic 関数でメッセージを表示し、プログラムを停止します。FINALROM ビルドでは、パルス振動が開始されません。

参照

VIBPulseState, VIB_StopPulse
