CARD_WriteAndVerifyFlashSector

構文

#include <nitro/card.h>

static inline BOOL CARD_WriteAndVerifyFlashSector(u32 dst, const void *src, u32 len);

引数

dst 転送先の FLASH アドレス
CARD_GetBackupSectorSize関数で得られる値の整数倍である必要があります。
デバイスごとのセクタサイズについては概要をご参照ください。
src 転送元のメモリアドレス
len 転送するデータのバイト長
CARD_GetBackupSectorSize関数で得られる値の整数倍である必要があります。
デバイスごとのセクタサイズについては概要をご参照ください。

返り値

処理が正しく成功してデータが一致していれば TRUE を、そうでないなら FALSE を返します。

説明

FLASH デバイスへのデータ書き込み、および書き込んだデータの比較をセクタ単位で行います。
この関数は処理が完了するまで制御を返しません。
処理が正しく成功してデータが一致していれば TRUE を、そうでない場合は FALSE を返します。
失敗時の詳細な結果値を取得するにはCARD_GetResultCode関数を呼び出してください。

この関数は、単に内部でCARD_EraseFlashSector関数とCARD_ProgramAndVerifyFlash関数を組み合わせて呼び出します。 詳細は各々の関数リファレンスをご参照ください。

この関数を使用する場合、事前にアクセスバスをCARD_LockBackup関数でロックしている必要があります。

内部動作

この関数は内部でPXIを介してARM7プロセッサと通信しています。
PXIの割り込みが有効でないと処理が完了しない点に注意してください。
