CARD_WaitBackupAsync

構文

#include <nitro/card.h>

BOOL CARD_WaitBackupAsync(void);

引数

なし

返り値

最後に実行された処理の結果が CARD_RESULT_SUCCESS であれば TRUE を、そうでなければ FALSE を返します。

説明

CARD バックアップデバイスへの非同期処理が実行中であれば完了までブロッキングします。
非同期処理が実行中でなければ、関数はすぐに制御を返します。
処理の結果が CARD_RESULT_SUCCESS であれば TRUE を、そうでなければ FALSE を返します。
詳細な結果値を取得するにはCARD_GetResultCode関数を呼び出してください。
