CARD_IsBackupEeprom

構文

#include <nitro/card.h>

static inline BOOL CARD_IsBackupEeprom(void);

引数

なし

返り値

指定されているバックアップデバイスが EEPROM であれば TRUE を、そうでないなら FALSE を返します。

説明

CARD_IdentifyBackup関数で指定されたバックアップデバイスの種類が EEPROM であるか判定します。

CARD_IdentifyBackup CARD_ReadEeprom CARD_WriteEeprom
