CARD_THREAD_PRIORITY_DEFAULT

定義

#include <nitro/card.h>

#define CARD_THREAD_PRIORITY_DEFAULT 4

説明

CARD非同期処理スレッドの優先度の初期値を表します。
このスレッドの優先度はCARD_SetThreadPriority関数で変更可能です。
CARD非同期処理スレッドの詳細についてはCARD_GetThreadPriority 関数の説明を参照ください。

参照

CARD_GetThreadPriority CARD_SetThreadPriority

履歴

2004/07/28 初版
