RTC_ConvertSecondToDateTime

構文

#include <nitro/rtc.h>
void RTC_ConvertSecondToDateTime( RTCDate* date , RTCTime* time, s64 sec );

引数

date 日付データを格納するバッファへのポインタ
time 時刻データを格納するバッファへのポインタ
sec 2000年1月1日午前0時0分0秒を0とした総経過秒数

返り値

なし。

説明

2000年1月1日午前0時0分0秒を0とした総経過秒数を RTCDate, RTCTime 型に変換します。
sec は 0 以上、3155759999 以下である必要があり、0 未満であれば 0 に、3155759999 より大きければ 3155759999 に、修正した後に結果を返します。

参照

RTC_ConvertDateTimeToSecond, RTCDate, RTCTime
