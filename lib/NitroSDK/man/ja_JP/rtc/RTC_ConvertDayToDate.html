RTC_ConvertDayToDate

構文

#include <nitro/rtc.h>
void RTC_ConvertDayToDate( RTCDate* date, s32 day );

引数

date 日付データを格納するバッファへのポインタ
day 2000年1月1日を 0 とした総経過日数

返り値

なし。

説明

2000年1月1日を 0 とした総経過日数を RTCDate> 型に変換します。
day は 0 以上、36524 以下である必要があり、0 未満であれば 0 に、36524 より大きければ 36524 に、修正した後に結果を返します。

参照

RTC_ConvertDateToDay, RTCDate
