RTC_GetDateTime

構文

#include <nitro/rtc.h>
RTCResult RTC_GetDateTime( RTCDate* date ,RTCTime* time );

引数

date 日付データを取得するバッファへのポインタ
time 時刻データを取得するバッファへのポインタ

返り値

要求が成功した場合に 0 を返します。
要求が失敗した場合には 0 以外を返します。

説明

リアルタイムクロックから日付データ及び時刻データを読み出します。
この関数はARM7に日付・時刻データ読み出し要求を発行し、ARM7からの応答を待って処理結果を返します。
RTC_GetDateTimeAsyncの同期版です。

注意

この関数は同期を取る為にPXI割り込みを待ちますので、PXI割り込みを禁止した状態で呼び出さないようにして下さい。
