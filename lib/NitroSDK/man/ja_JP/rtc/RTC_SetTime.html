RTC_SetTime

構文

#include <nitro/rtc.h>
RTCResult RTC_SetTime( const RTCTime* time );

引数

time 時刻データを格納したバッファへのポインタ

返り値

要求が成功した場合に 0 を返します。
要求が失敗した場合には 0 以外を返します。

説明

リアルタイムクロックに時刻データを書き込みます。
この関数はARM7に時刻データ書き込み要求を発行し、ARM7からの応答を待って処理結果を返します。
RTC_SetTimeAsyncの同期版です。

注意

書き込む時刻データには、存在しない時分秒を指定しないで下さい。
また、この関数は同期を取る為にPXI割り込みを待ちますので、PXI割り込みを禁止した状態で呼び出さないようにして下さい。
なお、アプリケーションが時刻を変更する事は禁止されていますので、本APIは開発途中のデバッグ時以外は使用しないで下さい。
