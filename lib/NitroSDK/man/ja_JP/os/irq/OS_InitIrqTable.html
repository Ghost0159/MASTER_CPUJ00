OS_InitIrqTable

構文

#include <nitro/os.h>

void OS_InitIrqTable( void );

引数

なし。

返り値

なし。

説明

IRQ割り込みテーブルの初期化を行ないます。

このテーブルは、IRQ割り込みが発生したときにジャンプする関数をIRQ割り込み要因別に登録しておくためのものです。殆どの割り込みに対し、何も行わない関数がセットされますが、DMA と TIMER 割り込みはOS 側で用意した割り込みハンドラを登録します。この割り込みだけは、ユーザがハンドラを登録した場合はＯＳのハンドラからコールされますが、最初は何も登録されていません。この違いはユーザが区別する必要はありません。従ってOS_InitIrqTable() を呼び出した後、ユーザからはすべての割り込みで何も行わない（あるいは何も行わない関数が呼び出される）ように見えます。

OS_InitIrqTable() は OS_Init() から呼び出されますので、OS_Init() を呼んでいる場合は呼ぶ必要はありません。
