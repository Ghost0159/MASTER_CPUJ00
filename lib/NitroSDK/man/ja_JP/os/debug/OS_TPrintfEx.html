OS_TPrintfEx

構文

#include <nitro/os.h>

void OS_TPrintfEx( const char *fmt, ... );

引数

fmt 出力書式文字列(フォーマット文字列)
... 可変個引数

返り値

なし。

説明

文字列を表示します。OS_TPrintf()を機能拡張したものであり、標準でない変換指定子に対応しています。 現在対応しているものについてはOS_VSNPrintfEx()を参照してください。

この関数は weak シンボルでコンパイルされますので、ユーザが独自で関数を定義すれば、それで上書きされます。

これはデバッグ用の関数です。最終ROM版(FINALROM)ライブラリでは出力動作を行ないません。

参照

OS_TPrintf, OS_VSNPrintfEx
