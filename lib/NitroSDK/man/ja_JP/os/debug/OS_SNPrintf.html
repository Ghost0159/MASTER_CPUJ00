OS_SNPrintf

構文

#include <nitro/os.h>

int OS_SNPrintf( char *dst, size_t len, const char *fmt, ... );

引数

dst 出力文字列の格納先バッファ
len 出力文字列の格納先バッファのサイズ
fmt 出力書式文字列(フォーマット文字列)
... 可変個引数

返り値

常に、書式文字列を正しく出力した場合の文字数を返します。この文字数に終端文字 '\0' は含まれません。

説明

軽量版の書式文字列関数です。

引数形式が異なることを除いて、 OS_VSNPrintf() と等価です。

この関数は weak シンボルでコンパイルされますので、ユーザが独自で関数を定義すれば、それで上書きされます。

参照

OS_SPrintf OS_VSPrintf OS_VSNPrintf
