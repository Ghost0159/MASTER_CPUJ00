OS_Printf

構文

#include <nitro/os.h>

void OS_Printf( const char *fmt, ... );

引数

fmt 出力書式文字列(フォーマット文字列)
... 可変個引数

返り値

なし。

説明

文字列を表示します。フォーマット文字列(出力書式文字列)に従い可変個の引数を与えることが可能です。

この関数は weak シンボルでコンパイルされますので、ユーザが独自で関数を定義すれば、それで上書きされます。

これはデバッグ用の関数です。最終ROM版(FINALROM)ライブラリでは出力動作を行ないません。

(例)
OS_Printf( "variable test = %d\n", test );

注意

OS_Printfは多くのスタックを消費します。割り込み中やコールバック実行中などに使用されるIRQスタックはそれほど大きくありませんので、 IRQ割り込み中の深いネストの中ではできるだけ軽量版のOS_TPrintfを使用するようにしてください。デバッグ出力をするとフリーズするような場合には、OS_TPrintfへ置き換えると状況が改善する場合があります。
ただし、OS_TPrintfでは浮動小数の出力がサポートされていません。
