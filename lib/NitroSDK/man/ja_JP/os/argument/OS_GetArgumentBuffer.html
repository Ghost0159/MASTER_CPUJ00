OS_GetArgumentBuffer

構文

#include <nitro/os.h>
const char* OS_GetArgumentBuffer( void );

引数

なし。

返り値

現在の引数バッファへのポインタです。

説明

現在設定されている、引数バッファへのポインタを取得します。

デフォルトでは、システム内で持っている引数バッファ領域( buryarg ツールによって埋め込まれます) が設定されていますが、ユーザが OS_SetArgumentBuffer() で変更することが出来ます。

FINALROM ビルドでは、必ず NULL を返します。

参照

OS_GetArgv, OS_GetArgc, OS_GetOpt, buryarg ツール,
OS_SetArgumentBuffer, OS_ConvertToArguments
