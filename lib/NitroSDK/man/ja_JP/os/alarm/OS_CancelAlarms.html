OS_CancelAlarms

構文

#include <nitro/os.h>
void OS_CancelAlarms( u32 tag );

引数

tag アラームをキャンセルしたいグループのタグ値

返り値

なし。

説明

指定のタグ値 tag を持つアラームを取り消します。
取り消すアラームにはワンショットアラーム、周期的アラームの区別はありません。
タグ値「0」はシステムで予約されていますので、ユーザは使用できません。

参照

OS_InitAlarm, OS_CancelAlarm, OS_SetAlarmTag, OS_CancelAllAlarms
