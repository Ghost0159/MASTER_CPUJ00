OS_Exit

構文

#include <nitro/os.h>

void OS_Exit( int status );

引数

staus 終了ステータス

返り値

なし。

説明

終了する旨を表示し、CPU を HALT 状態にしてループします。この関数から戻ることはありません。(内部では OS_Terminate() を呼んでいます。)

表示する際に、status で与えられたステータスを表示します。

loadrun や nitrorun を用いている場合、この関数が呼ばれ、終了する旨が表示されることでコンソール上のプロンプトに制御が戻ります。

参照

OS_Terminate, loadrun, nitrorun
