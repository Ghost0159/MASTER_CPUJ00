OS_EnableInterrupts

構文

#include <nitro/os.h>

OSIntrMode OS_EnableInterrupts( void );

引数

なし。

返り値

この関数を呼ぶ前の状態です。
OSIntrMode_Irq 列挙型

OS_INTRMODE_IRQ_DISABLE IRQディセーブル(禁止)
OS_INTRMODE_IRQ_ENABLE IRQイネーブル(許可)

の値が返ります。

説明

IRQ 割り込みを許可します。
これはすべての割り込みのマスターイネーブルです。IRQ割り込み要因のイネーブルを個別にセットしていても、この関数でIRQ が許可になっていないと割り込みは発生しません。

返り値は、この関数を呼ぶ以前の状態となります。

内部動作

CPSRレジスタ(current program status register)の IRQ割り込みの制御ビットをリセット(リセットがイネーブル)にしています。
