OS_SetVAlarmTag

構文

#include <nitro/os.h>
void OS_SetVAlarmTag( OSVAlarm* alarm, u32 tag );

引数

alarm タグを設定するVカウントアラーム構造体へのポインタ
tag 設定するタグ値

返り値

なし。

説明

Vカウントアラームにタグを設定します。
これは、OS_CancelVAlarms() で、指定のタググループのVカウントアラームをキャンセルするのに使用されます。

タグ値「0」はシステムで予約されていますので、ユーザが使用することは出来ません。

参照

OS_InitVAlarm, OS_CancelVAlarms
