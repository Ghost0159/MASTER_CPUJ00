OS_GetThreadParameter

構文

#include <nitro/os.h>
void* OS_GetThreadParameter( const OSThread* thread );

引数

thread スレッド構造体へのポインタ

返り値

指定のスレッドのユーザパラメータの値。

説明

スレッドのユーザパラメータを取得します。

ユーザパラメータは、ユーザが自由に使用できる領域です。全てのスレッドにこのための１つの void* 型領域が与えられており、この値をどのように使用しても構いません。システム側でこの値を変更したり参照することはありません。

ユーザパラメータは OS_SetThreadParameter() で設定されます。
