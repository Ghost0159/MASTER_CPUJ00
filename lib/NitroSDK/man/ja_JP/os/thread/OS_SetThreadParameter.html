OS_SetThreadParameter

構文

#include <nitro/os.h>
void OS_SetThreadParameter( OSThread* thread, void* parameter );

引数

thread スレッド構造体へのポインタ
parameter 設定するパラメータ値

返り値

なし。

説明

スレッドにユーザパラメータを設定します。

ユーザパラメータは、ユーザが自由に使用できる領域です。全てのスレッドにこのための１つの void* 型領域が与えられており、この値をどのように使用しても構いません。システム側でこの値を変更したり参照することはありません。

設定したユーザパラメータを取得するには、OS_GetThreadParameter() を使用してください。
