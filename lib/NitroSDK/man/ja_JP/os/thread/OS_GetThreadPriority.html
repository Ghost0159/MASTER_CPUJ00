OS_GetThreadPriority

構文

#include <nitro/os.h>
u32 OS_GetThreadPriority( const OSThread* thread );

引数

thread 優先度を取得するスレッド構造体へのポインタ

返り値

指定のスレッドの優先度。

説明

スレッドの優先度を返します。
スレッド優先度は 0〜31で、0がもっとも優先度の高いスレッドとなります。

参照

OS_InitThread, OS_SetThreadPriority
