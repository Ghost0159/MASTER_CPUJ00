SND_GetPlayerLocalVariable

構文

#include <nitro/snd.h>
s16 SND_GetPlayerLocalVariable( int playerNo, int varNo );

引数

playerNo プレイヤー番号です。値の範囲は0〜15です。
varNo 変数番号です。値の範囲は0〜15です。

返り値

シーケンスローカル変数の値を返します。

説明

シーケンスローカル変数の値を取得します。

シーケンスローカル変数の初期値は-1です。変数の初期化はシステム起動時及び、 シーケンスのスタート時に行われます。

この関数は、ARM7コマンド関数とは非同期に実行されます。 同期を取るためには、SND_IsFinishedCommandTag関数などで、 ARM7コマンドの処理が完了したことを確認する必要があります。
