SND_SetWaveDataAddress

構文

#include <nitro/snd.h>
void SND_SetWaveDataAddress(
        SNDWaveArc* waveArc,
        int index,
        const SNDWaveData* address );

引数

waveArc 波形アーカイブのポインタです。
index 波形データのインデックス番号です。
address 波形データのアドレスです。

返り値

なし。

説明

波形アーカイブ中の波形データのアドレスを更新します。 波形データは、SNDWaveData型で指定します。

addressにNULLを指定すると、 その波形インデックスに対する発音は行わなくなります。

SNDWaveData
