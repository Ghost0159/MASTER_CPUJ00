SND_GetFirstInstDataPos

構文

#include <nitro/snd.h>
SNDInstPos SND_GetFirstInstDataPos( const SNDBankData* bank );

引数

bank バンクデータのポインタです。

返り値

インストデータ走査のための位置情報を返します。

説明

バンクデータ中のインストデータ走査のための位置情報を取得します。

取得した位置情報を SND_GetNextInstData関数に渡すことで、 SNDInstData型のインストデータを、順番に取得することができます。 

    SNDInstPos pos = SND_GetFirstInstDataPos( bank );
    SNDInstData inst;
    
    while( SND_GetNextInstData( bank, &inst, &pos ) ) {
        // inst に対する処理を行う
    }
