SVC_GetCRC16

構文

#include <nitro/os/common/systemCall.h>
u16 SVC_GetCRC16( u32 start, const void *datap, u32 size );

引数

start 初期値
datap データアドレス
size サイズ(バイト数)

返り値

算出した CRC-16 の値です。

説明

CRC-16 を算出します。
データアドレス datap は2Byte境界である必要があります。
サイズ size は 2の倍数である必要があります。

CRCとはCyclic Redundancy Check (巡回冗長検査) の略で、データの誤りを検出するための手法です。

参照

なし。
