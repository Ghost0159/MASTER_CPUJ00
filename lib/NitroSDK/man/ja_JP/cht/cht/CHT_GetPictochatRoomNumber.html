CHT_GetPictochatRoomNumber

構文

#include <nitro/cht.h>

int CHT_GetPictochatRoomNumber( const WMBssDesc* pWmBssDesc );

引数

pWmBssDesc 調査するビーコン情報へのポインタを指定します。

返り値

ピクトチャットの親機が開いている部屋の番号( 0 〜 3 )を返します。 引数に指定したビーコン情報が異常であった場合には -1 を返します。

説明

ピクトチャットの親機ビーコン情報から、その親機が開いている部屋の番号を調べます。0 〜 3 がそれぞれ チャットルーム A 〜 D に対応しています。
当関数にて部屋番号を調べる前に、CHT_IsPictochatParent関数を用いて親機ビーコン情報がピクトチャットのビーコン情報であることを必ず確認するようにして下さい。 ピクトチャットのビーコン情報以外を引数に与えた場合の返り値は不定です。

CHT_IsPictochatParent, WMBssDesc
