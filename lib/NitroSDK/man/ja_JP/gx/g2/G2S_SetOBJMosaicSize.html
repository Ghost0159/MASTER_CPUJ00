G2S_SetOBJMosaicSize

構文

#include <nitro/gx/g2.h>

void G2S_SetOBJMosaicSize(int hSize, int vSize);

引数

hSize モザイクの横幅 (0 <= ｈSize <= 15)
vSize モザイクの縦幅 (0 <= ｖSize <= 15)

返り値

なし。

説明

サブ２DエンジンのOBJにかかるモザイクのサイズを決定します。ただし、モザイクサイズ値が0の場合は通常表示になります。
