G2_SetBlendBrightness

構文

#include <nitro/gx/g2.h>

void G2_SetBlendBrightness(
    int plane /* GXBlendPlaneMask */,
    int brightness
);

引数

plane 輝度変更対象の平面を選択（GXBlendPlaneMask型の値でorをとったもの）
brightness 輝度変更のための係数を指定します(-16 <= brightness <= 16)

返り値

なし。

説明

planeで指定した面の輝度を変更します。brightnessが正なら輝度アップ、負なら輝度ダウンです。GXBlendPlaneMask型についてはG2_SetBlendAlphaを参照してください。

参照

G2_BlendNone, G2_SetBlendAlpha, G2_SetBlendBrightnessExt, G2_ChangeBlendAlpha, G2_ChangeBlendBrightness
