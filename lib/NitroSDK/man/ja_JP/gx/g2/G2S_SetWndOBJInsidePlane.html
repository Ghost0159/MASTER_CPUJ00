G2S_SetWndOBJInsidePlane

構文

#include <nitro/gx/g2.h>

void G2S_SetWndOBJInsidePlane(int wnd /* GXWndPlaneMask */, BOOL effect );

引数

wnd ウィンドウの指定
effect カラー特殊効果(αブレンディングと輝度調整)の有無

返り値

なし。

説明

サブ２DエンジンのOBJウィンドウの内側に表示される面を指定します。GXWndPlaneMask型の値の論理和をとってください。G2S_SetWnd0InsidePlaneを参照のこと。

参照

G2S_SetWnd0InsidePlane, G2S_SetWnd1InsidePlane, G2S_SetWndOutsidePlane, G2S_GetWndOBJInsidePlane
