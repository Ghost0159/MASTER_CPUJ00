G2S_SetBlendBrightness

構文

#include <nitro/gx/g2.h>

void G2S_SetBlendBrightness(
    int plane /* GXBlendPlaneMask */,
    int brightness
);

引数

plane 輝度変更対象の平面を選択（GXBlendPlaneMask型の値でorをとったもの）
brightness 輝度変更のための係数を指定します(-16 <= brightness <= 16)

返り値

なし。

説明

サブ２Dエンジンで、planeで指定した面の輝度を変更します。brightnessが正なら輝度アップ、負なら輝度ダウンです。GXBlendPlaneMask型についてはG2S_SetBlendAlphaを参照してください。

参照

G2S_BlendNone, G2S_SetBlendAlpha, G2S_SetBlendBrightnessExt, G2S_ChangeBlendAlpha, G2S_ChangeBlendBrightness
