G2_GetBG2ControlLargeBmp

構文

#include <nitro/gx/gx_bgcnt.h>
static inline GXBg23ControlDCBmp G2_GetBG2ControlLargeBmp( void );

引数

なし。

返り値

BG2の設定。

説明

大画面256色ビットマップBGの場合のBG2の設定をGXBg2ControlLargeBmp構造体として取得します。
各メンバについてはG2_SetBG2ControlLargeBmpを参照してください。

この関数はBG2が大画面256色ビットマップBGに設定されている状態で使用する必要があります。

GXBg2ControlLargeBmpの定義は以下のようになっております。

typedef union
{
    u16 raw;
    struct
    {
        u16 priority    :2;
        u16 _reserve1   :4;
        u16 mosaic      :1;
        u16 _reserve2   :6;
        u16 areaOver    :1;
        u16 screenSize  :2;
    };
}
GXBg2ControlLargeBmp;
