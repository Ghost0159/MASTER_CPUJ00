G2_SetWnd0InsidePlane

構文

#include <nitro/gx/g2.h>

void G2_SetWnd0InsidePlane(int wnd /* GXWndPlaneMask */, BOOL effect );

引数

wnd ウィンドウに表示される面の指定
effect カラー特殊効果(αブレンディングと輝度調整)の有無

返り値

なし。

説明

ウィンドウ0の内側に表示される面を指定します。GXWndPlaneMask型の値の論理和をとってください。

GX_WND_PLANEMASK_NONE どの面も指定しません。
GX_WND_PLANEMASK_BG0 BG0を指定します。
GX_WND_PLANEMASK_BG1 BG1を指定します。
GX_WND_PLANEMASK_BG2 BG2を指定します。
GX_WND_PLANEMASK_BG3 BG3を指定します。
GX_WND_PLANEMASK_OBJ OBJを指定します。


以下にGXWndPlaneMask型の型定義を示しておきます。

typedef enum
{
    GX_WND_PLANEMASK_NONE = 0x0000,
    GX_WND_PLANEMASK_BG0 = 0x0001,
    GX_WND_PLANEMASK_BG1 = 0x0002,
    GX_WND_PLANEMASK_BG2 = 0x0004,
    GX_WND_PLANEMASK_BG3 = 0x0008,
    GX_WND_PLANEMASK_OBJ = 0x0010
}
GXWndPlaneMask;
