G2S_SetBGMosaicSize

構文

#include <nitro/gx/g2.h>

void G2S_SetBGMosaicSize(int hSize, int vSize);

引数

hSize モザイクの横幅 ( 0 <= hSize <= 15)
vSize モザイクの縦幅 ( 0 <= vSize <= 15)

返り値

なし。

説明

サブ２Dエンジンの２D画面にかかるモザイクのサイズを決定します。G2S_BGxMosaicでモザイクを許可した場合にモザイクがかかるようになります。ただし、モザイクサイズ値が0の場合は通常表示になります。

参照

G2S_SetOBJMosaicSize, G2S_BG0Mosaic, G2S_BG1Mosaic, G2S_BG2Mosaic, G2S_BG3Mosaic
