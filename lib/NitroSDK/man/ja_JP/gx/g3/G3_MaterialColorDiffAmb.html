G3*_MaterialColorDiffAmb

構文

#include <nitro/gx/g3imm.h>
#include <nitro/gx/g3b.h>
#include <nitro/gx/g3c.h>

void G3_MaterialColorDiffAmb(RGB diffuse, GXRgb ambient, BOOL IsSetVtxColor);

// 以下ディスプレイリスト（コマンド列）作成用API
void G3B_MaterialColorDiffAmb(
    GXDLInfo* info,
    GXRgb diffuse,
    GXRgb ambient,
    BOOL IsSetVtxColor);

void G3C_MaterialColorDiffAmb(
    GXDLInfo* info,
    GXRgb diffuse,
    GXRgb ambient,
    BOOL IsSetVtxColor);

void G3BS_MaterialColorDiffAmb(
    GXDLInfo* info,
    GXRgb diffuse,
    GXRgb ambient,
    BOOL IsSetVtxColor);

void G3CS_MaterialColorDiffAmb(
    GXDLInfo* info,
    GXRgb diffuse,
    GXRgb ambient,
    BOOL IsSetVtxColor);

引数

info コマンドリスト情報
diffuse 拡散反射色
ambient 環境反射色
IsSetVtxColor 頂点カラーセットフラグ

返り値

なし。

説明

材質の拡散反射色(diffuse)と環境反射色(ambient)を設定します。また、頂点カラーセットフラグは、頂点カラーをセットしない(FALSE)か、拡散反射色を頂点カラーとしてセットする(FALSE以外)かを選択することが出来ます。
