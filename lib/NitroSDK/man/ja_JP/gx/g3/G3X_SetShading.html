G3X_SetShading

構文

#include <nitro/gx/g3x.h>

void G3X_SetShading(GXShading shading);

引数

shading トゥーンシェーディングかハイライトシェーディングかを選択します。

返り値

なし。

説明

トゥーンシェーディングを使用するか、ハイライトシェーディングを使用するかを選択します。G3_PolygonAttrでGX_POLYGONMODE_TOONがポリゴンモードとして選択された場合にどちらのシェーディングを使用するかを決定します。

GX_SHADING_TOON トゥーンシェーディング
GX_SHADING_HIGHLIGHT ハイライトシェーディング


以下にGXShading型の型定義を示しておきます。

typedef enum
{
    GX_SHADING_TOON = 0,
    GX_SHADING_HIGHLIGHT = 1
}
GXShading;
