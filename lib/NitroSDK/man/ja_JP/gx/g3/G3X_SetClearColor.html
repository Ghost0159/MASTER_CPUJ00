G3X_SetClearColor

構文

#include <nitro/gx/g3x.h>

void G3X_SetClearColor(GXRgb rgb, int alpha, int depth, int polygonID, BOOL fog);

引数

rgb クリアカラーのRGB成分
alpha クリアカラーのα成分(0 <= alpha <= 31)
depth クリアデプス値
polygonID アトリビュートバッファのポリゴンID初期値
fog アトリビュートバッファのフォグを有効にするかどうかの選択

返り値

なし。

説明

クリアカラーを設定します。
