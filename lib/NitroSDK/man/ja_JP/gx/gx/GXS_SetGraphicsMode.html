GXS_SetGraphicsMode

構文

#include <nitro/gx/gx.h>

void GXS_SetGraphicsMode(GXBGMode bgMode);

引数

bgMode BGモードを選択

返り値

なし。

説明

サブ2Dエンジンの表示モードを設定します。
bgModeは、BGモードを選択するための引数です。なお、メイン2Dエンジンでは指定可能であるGX_BGMODE_6はサブ2Dエンジンでは設定禁止となります。

GX_BGMODE_0 BG0=テキスト, BG1=テキスト, BG2=テキスト, BG3=テキスト
GX_BGMODE_1 BG0=テキスト, BG1=テキスト, BG2=テキスト, BG3=アフィン
GX_BGMODE_2 BG0=テキスト, BG1=テキスト, BG2=アフィン, BG3=アフィン
GX_BGMODE_3 BG0=テキスト, BG1=テキスト, BG2=テキスト, BG3=アフィン拡張BG
GX_BGMODE_4 BG0=テキスト, BG1=テキスト, BG2=アフィン, BG3=アフィン拡張BG
GX_BGMODE_5 BG0=テキスト, BG1=テキスト, BG2=アフィン拡張BG, BG3=アフィン拡張BG


以下にGXBGMode型の型定義を示しておきます。

typedef enum
{
    GX_BGMODE_0 = 0,
    GX_BGMODE_1 = 1,
    GX_BGMODE_2 = 2,
    GX_BGMODE_3 = 3,
    GX_BGMODE_4 = 4,
    GX_BGMODE_5 = 5,
    GX_BGMODE_6 = 6
}
GXBGMode;
