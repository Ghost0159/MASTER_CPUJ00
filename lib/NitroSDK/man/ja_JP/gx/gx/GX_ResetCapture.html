GX_ResetCapture

構文

#include <nitro/gx/gx_capture.h>

void GX_ResetCapture( void );

引数

なし

返り値

なし

説明

表示キャプチャコントロールレジスタのキャプチャイネーブルフラグをOFFに設定します。
キャプチャイネーブルレジスタはキャプチャ終了後、自動的にOFFになりますので、通常この関数で
明示的にリセットする必要はありません。

(注意1) 0ライン目で表示キャプチャが開始された後に、この関数でキャプチャを途中中断することはできません。キャプチャを解除する場合にはキャプチャの動作開始される前に行なってください。
