GX_GetDispCnt

構文

#include <nitro/gx/gx.h>

GXDispCnt GX_GetDispCnt( void );

引数

なし。

返り値

メイングラフィックスエンジンの表示コントロールレジスタの値を示すGXDispCnt型の値。

説明

メイングラフィックスエンジンの表示コントロールレジスタの値を取得します。

GXDispCnt型には以下の情報が含まれます。

bgMode BGモードの情報です。GX_SetGraphicsModeのGXBGMode型を参照してください。
bg0_2d3d BG0を3Dに割り当てるかどうかの情報です。GX_SetGraphicsModeのGXBG0As型を参照してください。
objMapChar キャラクタOBJの1D/2Dマッピングの選択フラグです。
GX_SetOBJVRamModeCharを参照してください。
objMapBmp ビットマップOBJのマッピング選択フラグです。
GX_SetOBJVRamModeBmpを参照してください。
blankScr 2D表示の強制ブランク設定です。このパラメータは2D表示のみを停止するものであり通常利用されません。表示のON/OFF制御にはGX_DispOff,GX_DispOnを使用してください。
visiblePlane 2D表示面の設定です。詳細はGX_SetVisiblePlaneのGXPlaneMask型を参照してください。
visibleWnd ウィンドウの表示設定です。詳細はGX_SetVisibleWndのGXWndMask型を参照してください。
dispMode 表示モードの設定です。詳細はGX_SetGraphicsModeのGXDispMode型を参照してください。
extObjMapChar キャラクタOBJのVRAM容量設定です。
GX_SetOBJVRamModeCharを参照。
extObjMapBmp ビットマップOBJのVRAM容量設定です。
GX_SetOBJVRamModeBmpを参照してください。
hBlankObjProc Hブランク中のOBJ処理設定を行ないます。詳細はGX_HBlankOBJProcを参照してください。
bgCharOffset BGキャラクタデータのオフセット値設定です。GX_SetBGCharOffsetのGXBGCharOffset型を参照してください。
bgScrOffset BGスクリーンデータのオフセット値設定です。
GX_SetBGScrOffsetのGXBGScrOffset型を参照してください。
bgExtPltt BG拡張パレット有効フラグです。GX_SetBankForBGExtPltt等のVRAMバンク設定関数中で設定されます。
objExtPltt OBJの拡張パレット有効フラグです。GX_SetBankForOBJExtPltt等のVRAMバンク設定関数中で設定されます。


以下にGXDispCnt型の型定義を示しておきます。

typedef union
{
        u32 raw;
        struct {
                u32             bgMode          :3;
                u32             bg0_2d3d        :1;
                u32             objMapChar      :1;
                u32             objMapBmp       :2;
                u32             blankScr        :1;
                u32             visiblePlane    :5;
                u32             visibleWnd      :3;
                u32             dispMode        :4;
                u32             extObjMapChar   :2;
                u32             extObjMapBmp    :1;
                u32             hBlankObjProc   :1;
                u32             bgCharOffset    :3;
                u32             bgScrOffset     :3;
                u32             bgExtPltt       :1;
                u32             objExtPltt      :1;
        };
}
GXDispCnt;
