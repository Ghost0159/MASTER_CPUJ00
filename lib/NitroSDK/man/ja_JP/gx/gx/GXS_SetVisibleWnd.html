GXS_SetVisibleWnd

構文

#include <nitro/gx/gx.h>

void GXS_SetVisibleWnd(int window);

引数

window GXWndMask型の論理和で表示ウィンドウを指定します。

返り値

なし。

説明

表示するウィンドウを選択します。GXWndMask型の値で論理和をとったものを引数とします。

GX_WNDMASK_NONE 何も指定しません。
GX_WNDMASK_W0 ウィンドウ0を指定します。
GX_WNDMASK_W1 ウィンドウ1を指定します。
GX_WNDMASK_OW OBJウィンドウを指定します。


以下にGXWndMask型の型定義を示しておきます。

typedef enum
{
    GX_WNDMASK_NONE = 0x00,
    GX_WNDMASK_W0 = 0x01,
    GX_WNDMASK_W1 = 0x02,
    GX_WNDMASK_OW = 0x04
}
GXWndMask;
