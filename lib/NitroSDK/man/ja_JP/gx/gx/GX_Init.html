GX_Init

構文

#include <nitro/gx/gx.h>

void GX_Init( void );

引数

なし

返り値

なし。

説明

グラフィックスのレジスタで3D関連以外の各レジスタを初期化します。内部でGX_SetPower(GX_POWER_ALL)をコールしています。

GX関数では内部でデフォルトでDMA 3を使用します。
使用するDMA番号を変更したい場合には、GX_InitExまたはGX_SetDefaultDMAで使用するDMA番号を設定してください。

この関数を呼び出す前に、OS_InitLock関数またはOS_Init関数が呼び出されている必要があります。

参照

G3X_Init, G3X_Reset, G3X_InitTable, GX_InitEx
