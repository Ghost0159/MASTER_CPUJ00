GX*_GetVisibleWnd

構文

#include <nitro/gx/gx.h>

int GX_GetVisibleWnd( void );
int GXS_GetVisibleWnd( void );

引数

なし。

返り値

GXWndMask型の論理和で表示ウィンドウを取得します。

説明

表示するウィンドウを取得します。GXWndMask型の値で論理和をとったものを引数とします。
GX_GetVisibleWnd関数ではメイングラフィックスエンジンの、GXS_GetVisibleWnd関数ではサブグラフィックスエンジンの設定を取得することができます。

GX_WNDMASK_NONE 何も指定しません。
GX_WNDMASK_W0 ウィンドウ0を指定します。
GX_WNDMASK_W1 ウィンドウ1を指定します。
GX_WNDMASK_OW OBJウィンドウを指定します。


以下にGXWndMask型の型定義を示しておきます。

typedef enum
{
    GX_WNDMASK_NONE = 0x00,
    GX_WNDMASK_W0 = 0x01,
    GX_WNDMASK_W1 = 0x02,
    GX_WNDMASK_OW = 0x04
}
GXWndMask;
