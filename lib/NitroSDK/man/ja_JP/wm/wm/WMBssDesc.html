WMBssDesc

定義

#include <nitro/wm.h>

typedef struct WMBssDesc
{
    u16         length;
    u16         rssi;
    u8          bssid[ WM_SIZE_BSSID ];
    u16         ssidLength;
    u8          ssid[WM_SIZE_SSID];
    u16         capaInfo;
    struct
    {
        u16     basic;
        u16     support;
    } rateSet;
    u16         beaconPeriod;
    u16         dtimPeriod;
    u16         channel;
    u16         cfpPeriod;
    u16         cfpMaxDuration;
    u16         gameInfoLength;
    u16         rsv;
    WMGameInfo  gameInfo;
} WMBssDesc;

要素

length この構造体全体に格納された有効なデータの、ハーフワード(u16)単位でのサイズです。
bssid 親機のBSS IDです。
この値は親機のMACアドレスであり機体ごとに固有です。
gameInfoLength gameInfoに格納された有効なデータバイト数です。
gameInfo 親機の指定した情報が格納されたWMGameInfo構造体です。
rssi ライブラリ内部で使用されます。
