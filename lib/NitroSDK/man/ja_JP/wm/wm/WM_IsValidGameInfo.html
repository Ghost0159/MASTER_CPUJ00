WM_IsValidGameInfo

構文

#include <nitro/wm.h>

static inline BOOL WM_IsValidGameInfo( const WMGameInfo* gameInfo, u16 gameInfoLength );

引数

gameInfo WMGameInfo 構造体へのポインタを指定します。
gameInfoLength WMGameInfo 構造体の長さを指定します。

返り値

WMGameInfo 構造体が有効なものであればTRUEを返します。

説明

WM_StartScan, WM_StartScanEx 関数で取得したビーコン情報中の WMGameInfo 構造体が有効な値かを確認します。 内部的には、gameInfoLength が十分なサイズであるかと、gameInfo->magicNumber が WM_GAMEINFO_MAGIC_NUMBER と等しいかをチェックします。

参照

WM_IsValidGameBeacon, WM_StartScan, WM_StartScanEx
