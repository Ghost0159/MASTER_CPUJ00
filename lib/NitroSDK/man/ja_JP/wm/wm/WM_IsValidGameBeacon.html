WM_IsValidGameBeacon

構文

#include <nitro/wm.h>

static inline BOOL WM_IsValidGameBeacon( const WMBssDesc* bssDesc )
{
    return WM_IsValidGameInfo( &bssDesc->gameInfo, bssDesc->gameInfoLength );
}

引数

bssDesc ビーコン情報へのポインタを指定します。

返り値

WMBssDesc 構造体に格納されているビーコン情報がDSワイヤレスプレイで接続する親機からのものであればTRUEを返します。

説明

WM_StartScan, WM_StartScanEx 関数で取得したビーコン情報がDSワイヤレスプレイ用の親機からのものであるかを確認します。 内部的に WM_IsValidGameInfo 関数を呼び出します。

参照

WM_IsValidGameInfo, WM_StartScan, WM_StartScanEx
