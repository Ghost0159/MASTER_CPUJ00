WM_CopyBssid

構文

#include <nitro/wm/ARM9/wm_api.h>

static inline
void WM_CopyBssid( const u8* src , u8* dst );
static inline
void WM_CopyBssid16( const u8* src , u8* dst );

引数

src コピー元のBSSIDへのポインタ。
WM_CopyBssid16 では、このアドレスは16bit境界でアラインメントされている必要があります。
dst コピー先のBSSIDへのポインタ。
WM_CopyBssid16 では、このアドレスは16bit境界でアラインメントされている必要があります。

返り値

なし。

説明

BSSIDのコピーをおこないます。
双方の比較アドレスが16bit境界でアラインメントされている場合に限り WM_CopyBssid16 が使用可能で、いくらかのパフォーマンス向上が得られます。
