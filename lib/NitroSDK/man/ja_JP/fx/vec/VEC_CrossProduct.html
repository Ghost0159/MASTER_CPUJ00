VEC_CrossProduct

構文

#include <nitro/fx/fx_vec.h>

void VEC_CrossProduct(
    const VecFx32 * a,
    const VecFx32 * b,
    VecFx32 * axb
);

引数

a ベクトルへのポインタ
b ベクトルへのポインタ
axb ベクトルaとベクトルbの外積が格納されるポインタ

返り値

なし

説明

VecFx32型ベクトルの外積を求めます。結果は*abに格納されます。
*a,*b,*abは同じベクトルへのポインタでも構いません。

参照

VecFx32, VEC_DotProduct
