MtxFx44

定義

#include <nitro/fx/fx.h>

typedef union
{
    struct
    {
        fx32 _00, _01, _02, _03;
        fx32 _10, _11, _12, _13;
        fx32 _20, _21, _22, _23;
        fx32 _30, _31, _32, _33;
    };
    fx32 m[4][4];
    fx32 a[16];
}
MtxFx44;

説明

fx32型の要素で構成される4x4行列です。
