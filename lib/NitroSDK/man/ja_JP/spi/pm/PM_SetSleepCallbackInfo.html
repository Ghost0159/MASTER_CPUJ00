PM_SetSleepCallbackInfo

構文

#include <nitro/spi.h>
void PM_SetSleepCallbackInfo(
　　　　 PMSleepCallbackInfo* info,
　　　　 PMSleepCallback callback,
　　　　 void* arg );

引数

info セットコールバック情報構造体へのポインタ
callback コールバック関数
arg コールバックの引数

返り値

なし。

説明

スリープモードのコールバック情報を設定します。

コールバック関数 callback と、その引数 arg を、 info で示したコールバック情報構造体 に登録します。そして、この info を指定することでスリープモードのコールバック登録を行います。

内部では、info のメンバに callback と arg をセットするだけの処理を行っています。
