MATH_CountTrailingZeros

構文

#include <nitro/math.h>
u32 MATH_CountTrailingZeros( u32 x );
u32 MATH_CTZ( u32 x );

引数

x 演算する数値

返り値

最下位から0のビットが連続でいくつあるかを返します。

説明

x を2進数32ビットで表現した際に、最下位ビットから0が何ビット続くかを求める関数です。0x80000000 は 31、0 は 32 となります。
MATH_CountLeadingZeros関数を使用したインライン関数になっています。

MATH_CTZ は MATH_CountTrailingZeros 関数の別名です。

なお、先頭から続く 0 を数える場合はMATH_CountLeadingZeros関数で求めることが可能です。
