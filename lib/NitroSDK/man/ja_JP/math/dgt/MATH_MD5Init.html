MATH_MD5Init

構文

#include <nitro/math/dgt.h>

static inline void MATH_MD5Init( MATHMD5Context *context );

引数

context 初期化する MD5 生成用のコンテキスト構造体へのポインタ。

返り値

なし。

説明

この関数は、MD5 を計算する際に使用する MATHMD5Context 型のコンテキスト構造体を初期化します。 MATH_MD5Update 関数・MATH_MD5GetHash 関数を使用する前に必ず呼び出すようにしてください。
また、初期化済みのコンテキスト構造体にこの関数を再適用しても構いません。その場合は、コンテキスト構造体は初期状態に戻ります。

MD5 はメッセージダイジェスト値を求めるためのアルゴリズムです。詳細は IETF (The Internet Engineering Task Force) が発行している RFC (Request For Comments) 1321 を参照してください。

参照

MATH_MD5Update, MATH_MD5GetHash
