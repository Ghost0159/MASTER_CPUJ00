MATH_MD5GetHash

構文

#include <nitro/math/dgt.h>

static inline void MATH_MD5GetHash( MATHMD5Context *context, void* digest );

引数

context MD5 生成用のコンテキスト構造体へのポインタ。
digest メッセージダイジェスト値を格納するメモリへのポインタ。

返り値

なし。

説明

この関数は、MD5 によるダイジェスト値を得ます。context 引数をあらかじめ MATH_MD5Init 関数で初期化した上で、 入力データを MATH_MD5Update 関数で与えておく必要があります。
生成されるメッセージダイジェスト値の長さは 128 bit、すなわち MATH_MD5_DIGEST_SIZE (= 16) バイトです。このサイズが格納できる領域を digest 引数に渡すようにしてください。
なお、この関数の内部で context を書き換えてしまいますので、同じ context を MATH_MD5Update 関数で使いつづけることはできません。

MD5 はメッセージダイジェスト値を求めるためのアルゴリズムです。詳細は IETF (The Internet Engineering Task Force) が発行している RFC (Request For Comments) 1321 を参照してください。

参照

MATH_MD5Init, MATH_MD5Update
