WBT_CancelCurrentCommand

構文

#include <nitro/wbt.h>

BOOL WBT_CancelCurrentCommand(u16 target);

引数

target コマンドをキャンセルする対象のaidビットマップ
子機の場合、親機である "1" のみ指定可能です。

返り値

現在処理中のコマンドが無ければ FALSE を、
あれば中断しそのコマンドコールバックを呼び出した後、TRUE を返します。

説明

現在発行中のコマンドをキャンセルします。
主にWBT_GetBlock関数を発行後、途中でキャンセル するのに使用します。
コマンドを要求された側では使用できません。

参照

WBT_RequestSync WBT_GetBlockInfo WBT_GetBlock WBT_PutUserData
