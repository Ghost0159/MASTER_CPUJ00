WBT_GetCurrentDownloadProgress

構文

#include <nitro/wbt.h>

BOOL WBT_GetCurrentDownloadProgress(u32 block_id, int aid, int *current_count, int *total_count);

引数

block_id 進捗を取得する受信中データブロックのID
aid 進捗を取得する受信対象のaid
current_count 受信済みパケット数を格納するポインタ
total_count 全体パケット数を格納するポインタ

返り値

指定されたブロックが現在受信中であれば進捗情報を格納して TRUE を、そうでないなら何もせず FALSE を返します。

説明

指定されたデータブロックが現在WBT_GetBlock関数で受信中であれば、
その進捗状況を「受信済みパケット数/全体パケット数」として取得します。

履歴
