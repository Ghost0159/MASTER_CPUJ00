WBT_MpParentSendHook

構文

#include <nitro/wbt.h>

int WBT_MpParentSendHook(void *sendbuf, int send_size);

引数

sendbuf WBTのためにWM_SetMPDataToPort関数などでMP通信する親機送信バッファ
send_size WBTに対して割り当てる、sendbufの親機パケットサイズ
この値はWBT_PACKET_SIZE_MINバイト以上である必要があります。

返り値

実際に送信すべきデータが格納されたバイト数を返します。

説明

WBT ライブラリの親機がMP通信で送信すべきデータを送信バッファに格納します。
送信バッファにはWBTライブラリのプロトコルに基づいたデータが適宜設定され、
これをMP送信のたびに繰り返すことによって各種コマンドが実行されます。

（注：この関数ではバッファにデータを格納するだけです。この後、WM_SetMPDataToPortなどでバッファを送信して下さい。）

任意の通信先からWBT_GetBlockによるデータ受信を要求されている場合、
設定によってはこの呼び出しの中でWBT_CMD_PREPARE_SEND_DATAコールバックが発生します。
