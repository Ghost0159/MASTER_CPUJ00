WBTGetBlockCallback

定義

#include <nitro/wbt.h>

typedef	struct
{
	u32 block_id;
	u32 recv_data_size;
	WBTRecvBufTable recv_buf_table;
	WBTPacketBitmapTable pkt_bmp_table;
} WBTGetBlockCallback;

説明

WBT_GetBlock関数完了時のコールバックで渡される構造体です。

各メンバの内容は以下の通りです。
block_id 受信完了したブロックIDです。
recv_data_size 受信データサイズです。
recv_buf_table WBT_GetBlock関数で指定した受信バッファテーブルです。
pkt_bmp_table WBT_GetBlock関数で指定した受信履歴記録用ビットマップバッファテーブルです。
