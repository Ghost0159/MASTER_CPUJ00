WBTBlockInfo

定義

#include <nitro/wbt.h>

typedef struct {
  u32 id;
  s32 block_size;
  u8  user_id[WBT_USER_ID_LEN];
} WBTBlockInfo;

説明

登録ブロックの情報を表す構造体です。
これらはWBT_GetBlockInfo関数で相手から受信します。

各メンバの内容は以下の通りです。
id ユーザが一意に割り当てたブロックIDです。
block_size データサイズです。
user_id ユーザ定義情報です。
指定されたデータがWBT_USER_ID_LENバイトに満たない分は0で埋められます。
