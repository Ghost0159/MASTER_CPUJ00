WBTCallback

定義

#include <nitro/wbt.h>

typedef void (*WBTCallback)( void *param );

説明

WBTライブラリのコールバック関数として要求される関数の形式です。
各引数の内容は以下の通りです。
param WBTCommand構造体へのポインタにキャストして結果情報を取得できます。

参照

WBT_InitParent WBT_InitChild WBT_RequestSync WBT_GetBlockInfo WBT_GetBlock WBT_PutUserData WBT_CancelCurrentCommand
WBTCommandType WBTCommand
