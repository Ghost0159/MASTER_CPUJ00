PRC_InitStrokes

構文

#include <nitro/prc.h>

void PRC_InitStrokes(
    PRCStrokes*     strokes,
    PRCPoint*       points,
    u32             size );

引数

strokes 入力点列構造体へのポインタ
points strokes で用いる PRCPoint の配列へのポインタ
size points の要素数

返り値

なし。

説明

入力点列を保持する PRCStrokes 構造体を初期化します。実際の座標を保持する PRCPoint の配列へのポインタと、その配列の要素数を指定します。

参照

PRC_InitInputPattern*, PRC_AppendPoint, PRC_AppendPenUpMarker, PRC_Clear, PRC_IsFull, PRC_IsEmpty
