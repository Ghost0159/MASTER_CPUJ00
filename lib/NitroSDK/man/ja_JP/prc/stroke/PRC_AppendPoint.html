PRC_AppendPoint

構文

#include <nitro/prc.h>

void PRC_AppendPoint(
    PRCStrokes* strokes,
    int         x,
    int         y );

引数

strokes 入力点列構造体へのポインタ
x strokes に追加する点の x 座標
y strokes に追加する点の y 座標

返り値

なし。

説明

strokes に点 (x, y) を追加します。

参照

PRC_InitStrokes, PRC_AppendPenUpMarker, PRC_Clear, PRC_IsFull

履歴

2004/06/23 初版
