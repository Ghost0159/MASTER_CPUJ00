PRCStrokes

定義

#include <nitro/prc.h>

typedef struct PRCStrokes
{
    PRCPoint            *points;
    int                 size;
    u32                 capacity;
} PRCStrokes;

要素

points 点列データを実際に格納する PRCPoint 構造体の配列へのポインタ
size 現在格納されている点の数
capacity 最大で格納できる点の数

説明

タッチパネルからの入力を座標値の列として保持します。ペンが上がったことを示すマーカーを途中にはさむことによって、複数の画の入力を保持することができます。

参照

PRCPoint, PRC_InitStrokes, PRC_AppendPoint, PRC_AppendPenUpMarker, PRC_Clear, PRC_IsFull
