PRC_GetInputPatternStrokes

構文

#include <nitro/prc.h>

void* PRC_GetInputPatternStrokes(
    PRCStrokes* strokes,
    const PRCInputPattern* input );

引数

strokes 点列を保存する PRCStrokes 構造体へのポインタ
input 入力パターンへのポインタ

返り値

なし。

説明

PRC_InitInputPattern* で整形された入力パターンから整形後の点列データを取り出します。結果の strokes 内では入力パターン構造体を構成する点列情報へのポインタが張られますので、strokes の点列データを書き換えてはいけません。

参照

PRC_InitInputPattern*
