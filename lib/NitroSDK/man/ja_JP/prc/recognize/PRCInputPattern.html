PRCInputPattern

定義

#include <nitro/prc.h>

typedef struct PRCInputPattern
{
    ...
} PRCInputPattern;

要素

各要素には直接アクセスしないでください。

説明

タッチパネルからの入力点列に、認識に用いるために整形（正規化・リサンプリング）を施したデータです。構成する各線分の長さや角度、バウンディングボックスなどもすでに計算されて格納されています。

参照

PRC_InitInputPattern*, PRC_GetRecognizedEntry*
