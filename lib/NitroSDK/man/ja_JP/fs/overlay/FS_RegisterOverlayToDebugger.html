FS_RegisterOverlayToDebugger

構文

#include <nitro/fs.h>

static inline void FS_RegisterOverlayToDebugger(const FSOverlayInfo *p_ovi, const FSFile *p_file);

引数

p_ovi 無視されます
p_file 無視されます

返り値

無し

説明

この関数はNITRO-SDK2.0RC2以前のバージョンではオーバーレイをデバッガへ登録するために必要でしたが、
NITRO-SDK2.0RC3以降のバージョンにおいては不要となりました。
旧互換のために残されており、呼び出しは単に無視されます。

この関数の以前の実装はFS_StartOverlay関数の内部で代替されています。
