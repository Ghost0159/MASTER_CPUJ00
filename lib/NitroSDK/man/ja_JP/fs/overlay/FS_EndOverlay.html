FS_EndOverlay

構文

#include <nitro/fs.h>

void FS_EndOverlay(FSOverlayInfo *p_ovi);

引数

p_ovi オーバーレイモジュールの情報が格納されたFSOverlayInfo構造体

返り値

なし

説明

オーバーレイモジュールの static uninitializer を実行します。
static uninitializer とは、静的記憶期間を持つオブジェクトの解体処理を指します。

本関数は、FS_StartOverlay()で開始されたオーバーレイモジュールの解放前に
一度だけ呼び出す必要があります。

内部動作

この関数は、C++ で記述されたオーバーレイの場合に
オーバーレイが含むスコープ内の全てのオブジェクトに対して
デストラクタを実行します。
