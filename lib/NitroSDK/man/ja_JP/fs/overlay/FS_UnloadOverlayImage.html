FS_UnloadOverlayImage

構文

#include <nitro/fs.h>

BOOL FS_UnloadOverlayImage(FSOverlayInfo *p_ovi);

引数

p_ovi オーバーレイモジュールの情報が格納されたFSOverlayInfo構造体

返り値

オーバーレイモジュールのイメージが正しくアンロードされれば TRUE を返し、そうでない場合は、FALSE を返します。

説明

オーバーレイモジュールのイメージをメモリからアンロードします。
現状、この関数は内部でFS_EndOverlay関数を呼び出すだけです。
オーバーレイモジュールをロードする場合、領域の競合する他の全てのロード済みのモジュールに対して
この関数を呼ぶ必要があります。

参照

FSOverlayInfo FS_LoadOverlayInfo FS_LoadOverlayImage FS_EndOverlay
