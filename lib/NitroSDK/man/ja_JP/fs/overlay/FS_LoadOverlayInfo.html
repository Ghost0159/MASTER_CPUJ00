FS_LoadOverlayInfo

構文

#include <nitro/fs.h>

BOOL FS_LoadOverlayInfo(FSOverlayInfo *p_ovi, MIProcessor target, FSOverlayID id);

引数

p_ovi オーバーレイモジュールの情報を格納するFSOverlayInfo構造体
target オーバーレイモジュール対象（ARM9プロセッサ/ARM7プロセッサ）
id オーバーレイモジュールのID

返り値

オーバーレイモジュールの情報が正しくロードされれば TRUE を返し、そうでない場合は、FALSE を返します。

説明

指定したプロセッサの指定した番号に関連付けられたオーバーレイモジュールに関する情報をロードします。
