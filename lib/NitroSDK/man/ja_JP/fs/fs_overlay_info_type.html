FSOverlayInfo

定義

#include <nitro/fs.h>

typedef	struct
{
/* private: */
    FSOverlayInfoHeader header;
    MIProcessor         target; /* target processor */
} FSOverlayInfo;


説明

オーバーレイモジュールの情報を格納する構造体です。
これらは各種オーバーレイ関数の引数として使用されます。


注意

この型の内部表現は将来の拡張で変更される可能性があります。
現在の実装におけるサイズやメンバに依存したプログラミングは避けてください。
