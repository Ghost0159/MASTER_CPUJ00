FS_SeekFileToBegin

構文

#include <nitro/fs.h>

inline BOOL FS_SeekFileToBegin( FSFile *p_file );

引数

p_file FSFile構造体のアドレス

返り値

ファイルポインタが正しく移動されれば TRUE を返し、そうでない場合は FALSE を返します。

説明

ファイルポインタをファイルの先頭へ移動します。
この関数は FS_SeekFile(p_file, 0, FS_SEEK_SET) と等価です。

参照

FSFile FSSeekFileMode FS_SeekFile FS_SeekFileToEnd
