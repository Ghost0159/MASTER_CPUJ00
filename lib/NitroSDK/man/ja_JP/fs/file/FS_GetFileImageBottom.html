FS_GetFileImageBottom

構文

#include <nitro/fs.h>

static inline u32 FS_GetFileImageBottom(const FSFile *p_file);

引数

p_file FSFile構造体のアドレス
これはファイル情報を保持している必要があります。

返り値

ファイルイメージ領域終端の、アーカイブ内におけるオフセット値を返します。

説明

ファイルイメージ領域終端の、アーカイブ内におけるオフセット値を返します。
FS_OpenFile関数などで開いたファイル以外に対してこの関数を呼び出したときの返り値は未定義です。
