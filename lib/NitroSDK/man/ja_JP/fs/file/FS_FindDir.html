FS_FindDir

構文

#include <nitro/fs.h>

BOOL FS_FindDir( FSFile *p_dir, const char *path );

引数

p_dir FSFile構造体のアドレス
path ディレクトリのパス名

返り値

ディレクトリリストが正しく開ければ TRUE を, そうでないなら FALSE を返します。

説明

指定したパス名でディレクトリリストを開きます。
パス名の形式についてはFS_OpenFile関数の説明を参照下さい。

開いたディレクトリリストを使用してFS_ReadDir関数を呼び出すことにより、
ディレクトリ内のファイルエントリをFSDirEntry構造体として順に列挙することができます。

注意

この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
