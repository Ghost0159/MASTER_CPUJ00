FS_IsDir

構文

#include <nitro/fs.h>

static inline BOOL FS_IsDir(volatile const FSFile *p_file);

引数

p_file FSFile構造体のアドレス

返り値

ディレクトリ位置情報を格納しているなら TRUE を、そうでないなら FALSE を返します。

説明

指定したファイルが現在ディレクトリ位置情報を格納しているか判定します。
ディレクトリ位置情報は、FS_FindDir 関数または FS_SeekDir 関数で格納され、
FS_OpenFile 関数、FS_OpenFileFast 関数、および FS_CloseFile 関数で破棄されます。

参照

FSFile FS_FindDir FS_SeekDir
