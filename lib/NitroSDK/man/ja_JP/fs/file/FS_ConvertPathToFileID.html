FS_ConvertPathToFileID

構文

#include <nitro/fs.h>

BOOL FS_ConvertPathToFileID( FSFileID *p_file_id, const char *path );

引数

p_file_id 結果を格納するFSFileID構造体のアドレス
path ファイルのパス名

返り値

ファイルIDが正しく取得できれば TRUE を, そうでないなら FALSE を返します。

説明

指定したパス名からファイルIDを取得します。
このファイルIDを指定してFS_OpenFileFast関数を呼ぶことでファイルをオープンできます。

パス名の形式についてはFS_OpenFile関数の説明を参照下さい。

注意

この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。


参照
