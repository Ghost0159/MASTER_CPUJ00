FS_IsFile

構文

#include <nitro/fs.h>

static inline BOOL FS_IsFile(volatile const FSFile *p_file);

引数

p_file FSFile構造体のアドレス

返り値

ファイル情報を格納しているなら TRUE を、そうでないなら FALSE を返します。

説明

指定したファイルが現在ファイル情報を格納しているか判定します。
ファイル情報は、FS_OpenFile 関数または FS_OpenFileFast 関数で格納され FS_CloseFile 関数で解放されます。

参照

FSFile FS_OpenFile FS_OpenFileFast FS_CloseFile
