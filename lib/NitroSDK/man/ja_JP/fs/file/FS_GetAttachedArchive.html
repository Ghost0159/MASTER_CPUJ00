FS_GetAttachedArchive

構文

#include <nitro/fs.h>

static inline FSArchive* FS_GetAttachedArchive(const FSFile *p_file);

引数

p_file FSFile構造体のアドレス
これはファイル情報またはディレクトリ位置を保持している必要があります。

返り値

指定のファイルに関連付けられたアーカイブへのポインタを返します。

説明

指定されたファイルに関連付けられたアーカイブへのポインタを返します。
もし存在すれば、そのアーカイブのポインタを返します。
ファイル情報またはディレクトリ位置のいずれも保持していないFSFile構造体に対して
この関数を呼び出したときに返される値は未定義です。

FSFile FSArchive
