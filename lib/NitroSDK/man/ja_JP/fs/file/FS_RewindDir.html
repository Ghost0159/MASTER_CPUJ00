FS_RewindDir

構文

#include <nitro/fs.h>

BOOL FS_RewindDir( FSFile *p_dir );

引数

p_dir FSFile構造体のアドレス

返り値

ディレクトリリストが正しく先頭に戻れば TRUE を返し、そうでない場合は、FALSE を返します。

説明

指定したディレクトリリストが現在保持している列挙位置を、先頭に戻します。
この関数は、先頭位置を指すFSDirPosの値を前もって保存しておき
それを使用してFS_SeekDir関数を呼び出すことと等価です。

注意

この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
