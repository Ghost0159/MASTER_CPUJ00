FS_FindArchive

構文

#include <nitro/fs.h>

FSArchive* FS_FindArchive(const char *name, int name_len);

引数

name 検索する名前を格納した文字列へのポインタ
name_len nameの文字列長

返り値

指定の名前で登録されたアーカイブがすでにある場合はそのポインタを、そうでないなら NULL を返します。

説明

指定された名前のアーカイブがファイルシステムに登録されているか検索します。
もし存在すれば、そのアーカイブのポインタを返します。
ここで返されるポインタは、登録のみされていてロードされていないアーカイブも含みます。
