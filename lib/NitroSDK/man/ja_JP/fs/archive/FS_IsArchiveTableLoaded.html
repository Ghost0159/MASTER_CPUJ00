FS_IsArchiveTableLoaded

構文

#include <nitro/fs.h>

static inline BOOL FS_IsArchiveTableLoaded(volatile const FSArchive *p_arc);

引数

p_arc FSArchive構造体のアドレス

返り値

アーカイブが現在テーブルをプリロード済みなら TRUE を、そうでないなら FALSE を返します。

説明

指定したアーカイブが現在FATおよびFNTのデータをプリロード済みか判定します。
テーブルをプリロードされている状態では、ディレクトリアクセスがメモリ経由で効率的になります。

FSArchive
FS_LoadArchiveTables
