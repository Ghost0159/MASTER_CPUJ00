FS_UnloadArchive

構文

#include <nitro/fs.h>

BOOL FS_UnloadArchive(FSArchive *p_arc);

引数

p_arc FSArchive構造体のアドレス

返り値

アーカイブが正しくファイルシステムからアンロードされた場合は TRUE を、そうでないなら FALSE を返します。

説明

アーカイブがをファイルシステムへロードされていれば、アンロードします。
現在アーカイブがコマンドを処理中であれば、そのコマンドの完了まで関数はブロッキングします。
また、アーカイブに処理待ちのコマンドがあれば、それらは全てキャンセルされます。
