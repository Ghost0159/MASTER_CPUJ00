FS_IsArchiveSuspended

構文

#include <nitro/fs.h>

static inline BOOL FS_IsArchiveSuspended(volatile const FSArchive *p_arc);

引数

p_arc FSArchive構造体のアドレス

返り値

アーカイブが現在サスペンド中なら TRUE を、そうでないなら FALSE を返します。

説明

指定したアーカイブが現在サスペンド中か判定します。
サスペンドされたアーカイブは、受け付けた全てのコマンドを保留して実行しません。

FSArchive
FS_ResumeArchive FS_SuspendArchive
