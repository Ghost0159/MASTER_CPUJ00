FS_COMMAND_IDLE

定義


FSResult FS_ARCHIVE_PROC_FUNC(FSFile *p_file, FSCommandType command /* = FS_COMMAND_IDLE */);

説明

アーカイブコマンド FS_COMMAND_IDLE の動作仕様を以下に示します。


このコマンドに渡される引数 
このコマンドに引数はありません。
p_file が指す FSFile 構造体は、一時的なものです。
このコマンドに要求される動作 
アーカイブが未処理の最後のコマンドを処理した後に必要な特別な処理を行います。
デフォルトの実装
何も行いません。
コマンドの結果
コマンドの戻り値は無視されます。
このコマンドが使用される関数
コマンドを使用する全てのFS関数
