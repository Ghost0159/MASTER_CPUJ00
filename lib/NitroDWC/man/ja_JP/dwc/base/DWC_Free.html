DWC_Free

構文

#include <dwc.h>

void DWC_Free( DWCAllocType name,
               void*        ptr,
               u32          size );

説明

DWC_SetMemFunc関数で設定したメモリ解放関数を用いてメモリを解放します。

引数

name ライブラリ開発用の情報です。0を指定してください。
ptr DWC_SetMemFunc関数で設定したメモリ確保関数で確保したメモリへのポインタ。
size ライブラリ開発用の情報です。0を指定してください。

返り値

なし

参照

DWC_SetMemFunc, DWC_Alloc, DWC_AllocEx, DWC_Realloc, DWC_ReallocEx
