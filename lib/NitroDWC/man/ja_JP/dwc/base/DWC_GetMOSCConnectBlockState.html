DWC_GetMOSCConnectBlockState

構文

#include <dwc.h>

BOOL DWC_GetMOSCConnectBlockState( void );

説明

マッチメイクブロックオプション設定時の、現在の状態を取得します。

このマッチメイクオプションは、サーバクライアントマッチメイクのサーバDSにおいて、1台のクライアントDSとの接続に成功した後、他の接続を受け付けないようにするものです。 DWC_SetMatchingOption関数で設定することができます。

本関数では、クライアントDSからの接続をブロック中かどうかを取得します。

引数

なし

返り値

TRUE クライアントDSからの接続をブロック中。
FALSE クライアントDSからの接続をブロック中でないか、このオプションが設定されていない。

参照

DWC_SetMatchingOption
