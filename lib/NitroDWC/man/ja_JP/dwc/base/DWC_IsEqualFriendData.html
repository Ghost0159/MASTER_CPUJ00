DWC_IsEqualFriendData

構文

#include <dwc.h>

BOOL DWC_IsEqualFriendData( const DWCFriendData* frienddata1, const DWCFriendData* frienddata2 );

説明

引数で渡された２つの友達情報が同じかどうかを判定します。

友達情報にはデータタイプが設定されており、DWC_GetFriendDataType関数でそれを取得することができます。本関数で同じ友達情報であると判定するのは、このデータタイプまで一致している必要があります。
なお、DWC_UpdateServersAsync関数を使った友達リストの同期処理で、リスト内に同じ友達の友達情報を複数発見した場合には、１つを残して自動的に削除されます。

引数

frienddata1 友達情報1の格納されたDWCFriendData構造体のバッファへのポインタ
frienddata2 友達情報2の格納されたDWCFriendData構造体のバッファへのポインタ

返り値

TRUE 同じ
FALSE 違う

参照

DWC_CreateExchangeToken, DWC_CreateFriendKeyToken, DWC_GetFriendDataType, DWC_UpdateServersAsync
