DWC_SetUserRecvCallback

構文


#include <dwc.h>

BOOL DWC_SetUserRecvCallback( DWCUserRecvCallback callback );

説明

データ受信完了コールバックを設定します。 本関数で受信完了コールバックを設定しておけば、
DWC_SendReliable / DWC_SendReliableBitmap / DWC_SendUnreliable / DWC_SendUnreliableBitmap 関数によって
送られてきたデータを受信した時点で、コールバックが呼び出されます。

引数

callback コールバック関数へのポインタ

返り値

TRUE コールバック関数設定成功
FALSE FriendsMatchライブラリ非動作中で設定失敗

参照

DWC_SetUserSendCallback, DWC_SendReliable, DWC_SendReliableBitmap, DWC_SendUnreliable, DWC_SendUnreliableBitmap, DWCUserRecvCallback
