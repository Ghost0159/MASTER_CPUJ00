DWC_GetMOMinCompState

構文

#include <dwc.h>

int DWC_GetMOMinCompState( u64* time );

説明

指定人数以下マッチメイク完了オプション設定時の、現在の状態を取得します。

このマッチメイクオプションは、ピアマッチメイクにおいて、一定時間を過ぎてもマッチメイクが完了しない場合に、DWC_ConnectToAnybodyAsync / DWC_ConnectToFriendsAsync関数で指定した人数以下でのマッチメイクを許可するものです。 DWC_SetMatchingOption関数で設定することができます。

本関数では、オプションとして設定したタイムアウト時間が過ぎたかどうかと、マッチメイク開始からの経過時間を取得することができます。

引数

time ピアマッチメイク開始からの経過時間（単位：msec）格納先ポインタ。必要なければNULLを指定してください。

返り値

0 オプションとして設定したタイムアウト時間内。
1 オプションとして設定したタイムアウト時間を過ぎており、指定人数以下でマッチメイクが完了する可能性がある。
-1 このマッチメイクオプションが設定されていない、もしくは無効と設定されている。

参照

DWC_SetMatchingOption, DWC_ConnectToAnybodyAsync, DWC_ConnectToFriendsAsync
