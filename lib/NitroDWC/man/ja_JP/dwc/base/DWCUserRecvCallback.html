DWCUserRecvCallback

構文

#include <dwc.h>

typedef void (*DWCUserRecvCallback)( u8 aid, u8* buffer, int size );

説明

DWC_SendReliable / DWC_SendReliableBitmap / DWC_SendUnreliable / DWC_SendUnreliableBitmap 関数によって
送られてきたデータを受信した時に呼び出されます。

本コールバック関数はDWC_SetUserRecvCallback関数によって設定することができます。

引数

aid 送信元のAID
buffer 受信したデータへのポインタ
size 受信したデータのサイズ

返り値

なし。

参照

DWC_SetUserRecvCallback, DWC_SendReliable, DWC_SendReliableBitmap, DWC_SendUnreliable, DWC_SendUnreliableBitmap
