DWC_CheckValidConsole

構文

#include <dwc.h>

BOOL DWC_CheckValidConsole( const DWCUserData* userdata );

説明

Wi-Fiコネクションでは、DSカードとDS本体をセットで扱います。この関数は、ユーザデータがDS本体とセットのユーザデータかどうかをチェックします。

アプリケーションでは、この関数を用いて、ログインしても問題ないかどうかをチェックしてください。 この関数を用いた具体的な処理については、プログラミングマニュアルを参照してください。

引数

userdata ユーザデータが格納されたDWCUserData構造体へのポインタ

返り値

TRUE 指定したユーザデータとDS本体は一致する。
FALSE 指定したユーザデータとDS本体が異なる。

参照

DWC_CreateUserData, DWC_ClearDirtyFlag, DWC_LoginAsync
