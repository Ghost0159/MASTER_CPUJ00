DWC_CheckFriendKey

構文

#include <dwc.h>

BOOL DWC_CheckFriendKey( const DWCUserData* userdata, u64 friend_key );

説明

friend_keyで指定された友達登録鍵の正当性を確認します。友達登録鍵を友達リストに入れる場合は、DWC_CreateFriendKeyToken関数を用いて、友達登録鍵から友達情報を作成してください。

正当でない場合は、アプリケーションでユーザに再度入力を促すか、入力エラーを表示してください。

引数

userdata ユーザデータが格納されたDWCUserData構造体へのポインタ
friend_key 友達登録鍵

返り値

TRUE 正当である
FALSE 不正である

参照

DWC_CreateFriendKeyToken , DWC_GetFriendKey
