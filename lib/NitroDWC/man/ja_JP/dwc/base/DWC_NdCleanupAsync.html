DWC_NdCleanupAsync

構文


#include <nd/dwc_nd.h>

BOOL DWC_NdCleanupAsync( DWCNdCleanupCallback callback )

説明

ダウンロードライブラリの終了処理を開始します。

ダウンロードライブラリ終了処理の完了時に、callbackで指定したコールバック関数が呼び出されます。
なお、コールバック関数が呼び出された時点では、ダウンロードライブラリで使用したソケットのクローズ処理は開始されていますが、完全にその処理が終了していないことがあります。

本関数をダウンロードライブラリが呼び出すコールバック関数内で呼び出さないでください。

引数

callback ダウンロードライブラリの解放完了時に呼び出されるコールバック関数へのポインタ

返り値

TRUE 成功。終了処理完了時にコールバック関数が呼び出されます。
FALSE ダウンロードライブラリが初期化されていません。
また、コールバック関数は呼び出されません。
