DWC_GetUdpPacketDropNum

構文

#include <dwc.h>

int DWC_GetUdpPacketDropNum( void );

説明

受信バッファ溢れのためにUDPパケットを受信できなかった回数を取得します。

この回数が多いようであれば、データ送信間隔やデータ送信量を調整するか、

DWC_InitFriendsMatch関数で設定する受信バッファサイズを大きくすることをお勧めします。

この回数は、アプリケーションが起動してからの総回数で、リセットされることはありません。

引数

なし

返り値

受信バッファ溢れのためにUDPパケットを受信できなかった回数

参照

DWC_InitFriendsMatch
