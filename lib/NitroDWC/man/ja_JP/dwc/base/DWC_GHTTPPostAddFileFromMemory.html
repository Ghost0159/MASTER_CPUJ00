DWC_GHTTPPostAddFileFromMemory

構文

#include <dwc.h>

BOOL DWC_GHTTPPostAddFileFromMemory( DWCGHTTPPost *post,
                                     const char *key,
                                     const char *buffer,
                                     int bufferlen,
                                     const char *filename,
                                     const char *contentType );

説明

HTTPサーバにアップロードするデータをDWCGHTTPPost型オブジェクトに追加します。 DWCGHTTPPost型オブジェクトは、データをHTTPサーバにアップロードするための情報単位で、DWC_GHTTPNewPost関数によって作成されます。

データはファイルとしてHTTPサーバにアップロードされます。 この呼び出しの中でデータはコピーされません。実際にアップロードを開始するときにデータが読み込まれまることに留意してください。

実際にデータのアップロードを開始するためには、DWC_PostGHTTPData関数、DWC_GetGHTTPDataEx2関数を使います。

引数

post データを追加するDWCGHTTPPost型オブジェクトへのポインタ
key 追加するデータの名前を表す文字列。NULLおよび空文字列""は指定できません。
buffer ファイルとしてアップロードするデータへのポインタ
bufferlen ファイルとしてアップロードするデータのバイト数
filename HTTPサーバに通知するファイル名。NULLおよび空文字列""は指定できません。
contentType ファイルのMIMEタイプ。NULLの場合は"application/octet-stream"となります。

返り値

TRUE データ追加成功
FALSE パラメータエラー、もしくはメモリ確保エラーのためデータ追加失敗

参照

DWC_GHTTPNewPost, DWC_PostGHTTPData, DWC_GetGHTTPDataEx2
