DWC_Ping

構文

#include <dwc.h>

BOOL DWC_Ping( u8 aid );

説明

接続されているホストのうち、特定のAIDに割り当てられたホストとの往復通信時間（単位：msec）を測定します。 測定結果は、DWC_SetUserPingCallback関数で設定したPingコールバック関数へ通知されます。

引数

aid 往復通信時間を測定する相手のAID

返り値

TRUE 正常終了
FALSE エラー発生中か無効なAID指定により失敗

参照

DWC_SetUserPingCallback, DWCUserPingCallback
