DWC_CanChangeFriendList

構文

#include <dwc.h>

BOOL DWC_CanChangeFriendList( void );

説明

DWC_InitFriendsMatch関数で指定した友達リストを編集しても良いかどうかを確認します。

この関数がFALSEを返す場合は、DWC_ProcessFriendsMatch関数を呼び出し、 本関数がTRUEを返すようになるまで待機してください。FALSEを返す状態が60秒程度続くとタイムアウトしてDWCライブラリ のエラーが発生します。

引数

なし

返り値

TRUE 編集可能。
FALSE 編集不可。

参照

DWC_InitFriendsMatch, DWC_UpdateServersAsync DWC_ProcessFriendsMatch
