DWC_StopSCMatchingAsync

構文

#include <dwc.h>

BOOL DWC_StopSCMatchingAsync(DWCStopSCCallback callback, void* param);

説明

サーバDSから現在進行中のサーバクライアントマッチメイクを締め切り、以後のクライアントDSからの接続要求を受け付けないようにします。

締め切り処理が完了し、以後のクライアントDSからの接続を拒否する状態になると、締め切り処理完了コールバックが呼び出されます。

締め切り処理を実行すると、接続可能な最大プレイヤー数を、現在接続しているプレイヤー数と等しい値にセットします。これにより、以後クライアントDSはこのサーバに接続することができなくなります。

また、締め切り後に接続済みのプレイヤーが抜けた場合は、その台数分のクライアントDSはサーバDSに接続が可能になります。

引数

callback 締め切り処理完了コールバック関数へのポインタ
param 締め切り処理完了コールバック用パラメータ

返り値

TRUE サーバクライアントマッチメイク締め切り処理の成功
FALSE サーバクライアントマッチメイク締め切り処理が実行できなかった
