DWC_IsValidAID

構文

#include <dwc.h>

BOOL DWC_IsValidAID( u8 aid );

説明

aidで指定されたAIDが、現在自分に接続中のホストに割り当てられているかどうかを調べます。

AIDはメッシュ型ネットワーク内での各DSの識別番号で、0〜（ネットワーク構成台数-1）の数値が用いられます。 例えば4人でマッチメイクを完了した場合なら、0, 1, 2, 3 の4台が存在し、AID = 1 の人がネットワークから抜ければ、残りは 0, 2, 3 となります。

引数

aid 調べたいAID

返り値

TRUE 指定されたAIDを割り当てられたホストと接続中である。
FALSE 指定されたAIDを割り当てられたホストは接続されていない。
