DWC_SetConnectApType

構文

#include <dwc.h>

void DWC_SetConnectApType( DWCApInfoType type );

説明

インターネットへの自動接続処理の際に接続するAPのタイプを指定します。 この場合の自動接続処理では、引数で指定されたAPタイプ以外のAPには接続されません。

本関数はDWC_InitInet/DWC_InitInetEx関数の呼び出しからDWC_ConnectInetAsync関数の呼び出しまでに呼び出す必要があります。

これはデバッグ用の関数です。 最終ROM版（FINALROM）ライブラリでは何も動作を行いません。

※　DWC_DebugConnectInetAsync関数とは一緒に呼び出さないようにしてください。

引数

type 接続するAPの種別を示す数値です。

DWC_APINFO_TYPE_USER0　　　　:　接続先1のアクセスポイント
DWC_APINFO_TYPE_USER1　　　　:　接続先2のアクセスポイント
DWC_APINFO_TYPE_USER2　　　　:　接続先3のアクセスポイント
DWC_APINFO_TYPE_USB　　　　　:　ニンテンドーWi-Fi USBコネクタ
DWC_APINFO_TYPE_SHOP 　　　　:　ニンテンドーWi-Fiステーション
DWC_APINFO_TYPE_FREESPOT 　　:　FREESPOT（フリースポット）
DWC_APINFO_TYPE_WAYPORT　　　:　Wayport(北米ホットスポット)
DWC_APINFO_TYPE_NINTENDOWFC　:　（予約）
DWC_APINFO_TYPE_NINTENDOSPOT :　nintendo spot(設定できません)
