DWC_InitGHTTP

構文

#include <dwc.h>

BOOL DWC_InitGHTTP( const char* gamename );

説明

GHTTPライブラリを初期化します。

GHTTPライブラリを使用する前に必ず一回呼び出してください。

この関数は複数回呼ぶ必要はありませんが、もし複数回呼び出した場合は、同じ回数だけDWC_ShutdownGHTTP関数を呼び出さなければ、GHTTPライブラリが確保したメモリが解放されずに残ってしまいます。

引数

gamename NULLを指定してください。過去との互換性のために残されています。

返り値

TRUE 必ずTRUEになります

参照

DWC_ShutdownGHTTP
