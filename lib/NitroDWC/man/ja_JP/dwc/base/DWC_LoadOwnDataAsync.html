DWC_LoadOwnDataAsync

構文


#include <dwc.h>

BOOL DWC_LoadOwnDataAsync( char* keys, void* param );

説明

この関数は過去との互換性のために残されています。 この関数では、DWC_SavePrivateDataAsync関数でセーブした自分のデータしかロードできません。

全く同じ内容のDWC_LoadOwnPrivateDataAsync関数を使用してください。

引数

keys ロードしたいデータのキー文字列へのポインタ
param コールバック用パラメータ

返り値

TRUE ロード開始
FALSE オフライン状態かエラー発生中でロードできない

参照

DWC_SavePublicDataAsync, DWC_SavePrivateDataAsync, DWC_LoadOwnPublicDataAsync, DWC_LoadOwnPrivateDataAsync
