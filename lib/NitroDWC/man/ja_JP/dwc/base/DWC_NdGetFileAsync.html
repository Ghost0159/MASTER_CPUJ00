DWC_NdGetFileAsync

構文


#include <nd/dwc_nd.h>

BOOL DWC_NdGetFileAsync( DWCNdFileInfo *fileinfo, char *buf, unsigned int buflen )

説明

ファイルのダウンロードを開始します。すでにダウンロードサーバと通信中の場合は失敗します。

ダウンロード処理の完了時に、DWC_NdInitAsync関数で指定したコールバック関数が呼び出されます。

ダウンロードしたデータを書き込むバッファは、ダウンロードするデータのサイズに関わらず最低 128バイトのバッファを用意する必要があります。

引数

fileinfo ダウンロードする対象のファイル情報へのポインタ
buf ダウンロードしたデータを書き込むバッファへのポインタ
buflen ダウンロードしたデータを書き込むバッファのサイズ(最低 128バイト)

返り値

TRUE 成功。ダウンロード終了時にコールバック関数が呼び出されます。
FALSE 失敗
DWC_GetLastErrorEx関数を使用してエラーの情報を取得してください

参照

DWC_NdInitAsync, DWC_GetLastErrorEx
