DWC_CloseConnectionsAsync

構文

#include <dwc.h>

int DWC_CloseConnectionsAsync( void );

説明

この関数は、過去の互換性のために残されています。DWC_CloseAllConnectionsHard関数を使用してください。

引数

なし

返り値

0 クローズ開始。クローズ処理が完了したらコールバックが呼ばれる。
1 接続ホスト数0なので、クローズ処理は行ったが、コールバックは呼ばれない。サーバクライアントマッチメイクのサーバDSの場合のみ。
-1 オフライン中、もしくはエラー発生時なので何もしなかった。

参照

DWC_CloseAllConnectionsHard

履歴
