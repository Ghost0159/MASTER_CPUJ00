DWC_SetGsProfileId

構文

#include <dwc.h>

void DWC_SetGsProfileId( DWCFriendData* frienddata,
                         int profile_id );

説明

frienddataで指定された友達情報にGSプロファイルIDを格納します。 デバッグ用にのみ使用してください。

引数

frienddata 友達情報の格納されたDWCFriendData構造へのポインタ
profile_id 格納するGSプロファイルID

返り値

なし
