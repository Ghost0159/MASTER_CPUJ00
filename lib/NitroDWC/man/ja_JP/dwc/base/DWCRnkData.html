DWCRnkData

定義


#include <dwc.h>

typedef struct{

    u32                 order;
    s32                 pid;
    s32                 score;
    DWCRnkRegion        region;
    u32                 lastupdate;
    u32                 size;
    void*               userdata;

} DWCRnkData;

説明

DWC_RnkResGetRow関数で取得するランキング情報を格納する構造体です。

order 順位
自分の順位以外の行には0が格納されています。ただし、トップランキングリスト取得モード（DWC_RNK_GET_MODE_TOPLIST）では、自分のランキング情報が入っている行でも、orderの値は0が格納されます。
pid GSプロファイルID
score スコア
region リージョンコード
lastupdate 前回の更新からの経過時間(分)
size userdataのサイズ(bytes)
userdata ユーザー定義データのバッファ
