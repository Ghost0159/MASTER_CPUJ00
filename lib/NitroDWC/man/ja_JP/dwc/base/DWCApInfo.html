DWCApInfo

定義


#include <dwc.h>

typedef struct DWCstApInfo
{
	DWCApInfoType    aptype;
	DWCApInfoArea    area;
	char             spotinfo[DWC_SIZE_SPOTINFO+1];
	char             essid[WM_SIZE_SSID+1];
	u8               bssid[WM_SIZE_BSSID];
} DWCApInfo;

説明

アクセスポイントの情報を格納するための構造体です。

aptype アクセスポイントの種類

DWC_APINFO_TYPE_USER0　　　　:　接続先1のアクセスポイント
DWC_APINFO_TYPE_USER1　　　　:　接続先2のアクセスポイント
DWC_APINFO_TYPE_USER2　　　　:　接続先3のアクセスポイント
DWC_APINFO_TYPE_USB　　　　　:　ニンテンドーWi-Fi USBコネクタ
DWC_APINFO_TYPE_SHOP 　　　　:　ニンテンドーWi-Fiステーション
DWC_APINFO_TYPE_FREESPOT 　　:　FREESPOT（フリースポット）
DWC_APINFO_TYPE_WAYPORT　　　:　Wayport(北米ホットスポット)
DWC_APINFO_TYPE_NINTENDOWFC　:　（予約）
DWC_APINFO_TYPE_NINTENDOSPOT :　nintendo spot
area ニンテンドーWi-Fiステーション/nintendo spotの設置地域
※aptypeがDWC_APINFO_TYPE_SHOP/DWC_APINFO_TYPE_NINTENDOSPOTの時のみ有効です。
※下記の全ての地域に設置されているとは限りません。

DWC_APINFO_AREA_JPN　　　:　日本
DWC_APINFO_AREA_USA　　　:　北米
DWC_APINFO_AREA_EUR　　　:　欧州
DWC_APINFO_AREA_AUS　　　:　豪州
DWC_APINFO_AREA_KOR　　　:　韓国

DWC_APINFO_AREA_UNKNOWN　:　(ニンテンドーWi-Fiステーション/nintendo spotではありません)
spotinfo ニンテンドーWi-Fiステーション/nintendo spotの場所情報
※aptypeがDWC_APINFO_TYPE_SHOP/DWC_APINFO_TYPE_NINTENDOSPOTの時のみ有効です。

各ニンテンドーWi-Fiステーション/nintendo spotでユニークな9桁の文字列です。
essid アクセスポイントのSSID
※aptypeがDWC_APINFO_TYPE_USER*の時のみ有効です。
bssid アクセスポイントのBSSID
