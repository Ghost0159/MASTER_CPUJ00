DWC_GetAIDList

構文

#include <dwc.h>

int DWC_GetAIDList( u8** aidList );

説明

接続中ホストのAIDリストへのポインタを取得します （コピーはしません ）。

AIDはメッシュ型ネットワーク内での各DSの識別番号で、0〜（ネットワーク構成台数-1）の数値が用いられます。 例えば4人でマッチメイクを完了した場合なら、0, 1, 2, 3 の4台が存在し、AID = 1 の人がネットワークから抜ければ、残りは 0, 2, 3 となります。
サーバクライアントマッチメイクにおいてはサーバDSのAIDが必ず0になります。

接続中ホストのAIDをビットマップとして取得したい場合は、DWC_GetAIDBitmap関数を使用してください。

引数

aidList AIDリストへのポインタ

返り値

自分も含めた接続中ホスト数。FriendsMatchライブラリ非動作中に本関数を呼び出した場合は0を返す。

参照

DWC_GetMyAID, DWC_GetAIDBitmap
