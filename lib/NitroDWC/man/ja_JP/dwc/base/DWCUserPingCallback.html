DWCUserPingCallback

構文

#include <dwc.h>

typedef void (*DWCUserPingCallback)( int latency, u8 aid );

説明

DWC_Ping関数での測定結果を通知するコールバック関数です。

本コールバック関数はDWC_SetUserPingCallback関数によって設定することができます。

引数

latency Pingに要した往復時間（ミリ秒）
aid Ping測定を行った相手のAID

返り値

なし。

参照

DWC_Ping, DWC_SetUserPingCallback
