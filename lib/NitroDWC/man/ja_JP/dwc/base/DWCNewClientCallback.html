DWCNewClientCallback

構文

#include <dwc.h>

typedef void (*DWCNewClientCallback)( int index, void* param );

説明

友達とのサーバクライアントマッチメイクで、今自分が属しているメッシュ型ネットワークに、新たに別のクライアントDSが接続を開始した時に呼び出されます。

本コールバック関数はDWC_SetupGameServer / DWC_ConnectToGameServerAsync関数によって設定することができます。

引数

index 新規接続クライアントDSの友達リストインデックス。新規接続クライアントDSが友達でなければ -1。
param DWC_SetupGameServer / DWC_ConnectToGameServerAsync関数で指定したコールバック用パラメータ

返り値

なし。

参照

DWC_SetupGameServer, DWC_ConnectToGameServerAsync
