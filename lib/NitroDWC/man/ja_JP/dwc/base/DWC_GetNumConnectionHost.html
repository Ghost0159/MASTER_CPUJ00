DWC_GetNumConnectionHost

構文

#include <dwc.h>

int DWC_GetNumConnectionHost( void );

説明

自分も含めた接続中ホスト数を取得します。

引数

なし

返り値

自分も含めた接続中ホスト数。マッチメイク途中の接続数もカウントされる。 FriendsMatchライブラリ非動作中に本関数を呼び出した場合は0を返す。

参照

DWC_InitFriendsMatch
