DWC_UpdateConnection

構文

#include <dwc.h>

BOOL DWC_UpdateConnection( void );

説明

アクセスポイントとの接続が正常に維持されているかを確認します。
インターネット接続を確立した後、本関数を毎ゲームフレーム程度の頻度で呼び出してください。ただし、FriendsMatchライブラリを使用する場合は、DWC_ProcessFriendsMatch関数で同等の処理を行なっているため、本関数を呼び出す必要はありません。

アクセスポイントから切断されると、TRUEが返されますので、DWC_GetLastErrorEx関数でエラー状態を取得し、必要に応じてエラーメッセージとともにエラーコードを表示してください。 メッセージについては、「Nitendo Wi-Fi Connection プログラミングガイドライン」をご確認ください。

引数

なし

返り値

TRUE アクセスポイントから切断された。
FALSE アクセスポイントとの接続が維持されている。

参照

DWC_GetLastErrorEx, DWC_ProcessFriendsMatch
