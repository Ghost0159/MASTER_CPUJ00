DWCNdError

定義


#include <nd/dwc_nd.h>

typedef enum {

	DWC_ND_ERROR_NONE,
	DWC_ND_ERROR_ALLOC,
	DWC_ND_ERROR_BUSY,
	DWC_ND_ERROR_HTTP,
	DWC_ND_ERROR_BUFFULL,
	DWC_ND_ERROR_DLSERVER,
	DWC_ND_ERROR_CANCELED,
	DWC_ND_ERROR_PARAM,
	DWC_ND_ERROR_FATAL

} DWCNdError;

説明

ダウンロードライブラリで発生したエラーの種類を表す列挙型です。

DWC_ND_ERROR_NONE 実行中の処理が完了した
DWC_ND_ERROR_ALLOC メモリ確保に失敗した
DWC_ND_ERROR_BUSY ダウンロード実行中にダウンロードを試みた
DWC_ERROR_HTTP HTTP通信にエラーが発生した
DWC_ND_ERROR_BUFFULL ダウンロードバッファが足りない
DWC_ND_ERROR_DLSERVER サーバがエラーコードを返してきた
DWC_ND_ERROR_CANCELED 処理のキャンセルが完了した
DWC_ND_ERROR_PARAM 不正なパラメータによるエラーが発生した
DWC_ND_ERROR_FATAL 未定義の致命的なエラーが発生した
