DWCMatchedCallback

構文

#include <dwc.h>

typedef void (*DWCMatchedCallback)( DWCError error,
                                    BOOL  cancel,
                                    void* param );

説明

DWC_ConnectToAnybodyAsync / DWC_ConnectToFriendsAsync関数呼出し後、マッチメイク処理が成功、エラー、キャンセルのいずれかの状態で完了した時に呼び出されます。

本コールバック関数はDWC_ConnectToGameServerAsync / DWC_SetupGameServer関数によって設定することができます。

引数

error DWCエラー種別。内容の詳細は、DWC_GetLastErrorExを参照してください。
cancel TRUE:キャンセルでマッチメイク終了
FALSE:成功かエラーでマッチメイク終了
param DWC_ConnectToAnybodyAsync / DWC_ConnectToFriendsAsync関数で指定したコールバック用パラメータ

返り値

なし。

参照

DWC_ConnectToAnybodyAsync, DWC_ConnectToFriendsAsync, DWC_GetLastErrorEx
