NNS_SndArcGet*Count

構文

#include <nnsys/snd.h>
u32 NNS_SndArcGetSeqCount( void );
u32 NNS_SndArcGetSeqArcCount( void );
u32 NNS_SndArcGetBankCount( void );
u32 NNS_SndArcGetWaveArcCount( void );
u32 NNS_SndArcGetGroupCount( void );
u32 NNS_SndArcGetStrmCount( void );

引数

なし。

返り値

各種、サウンドアーカイブ中のデータの個数を返します。

説明

各種、サウンドアーカイブ中のデータの個数を取得します。

NNS_SndArcGetSeqCount関数は、シーケンスの個数を取得します。

NNS_SndArcGetSeqArcCount関数は、シーケンスアーカイブの個数を取得します。

NNS_SndArcGetBankCount関数は、バンクの個数を取得します。

NNS_SndArcGetWaveArcCount関数は、波形アーカイブの個数を取得します。

NNS_SndArcGetGroupCount関数は、グループの個数を取得します。

NNS_SndArcGetStrmCount関数は、ストリームの個数を取得します。

参照

なし。
