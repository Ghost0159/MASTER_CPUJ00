NNS_SndPlayerWriteVariable

構文

#include <nnsys/snd.h>
BOOL NNS_SndPlayerWriteVariable(
        NNSSndHandle* handle,
        int varNo,
        s16 var );

引数

handle サウンドハンドルのポインタです。
varNo 変数番号です。値の範囲は、0〜15です。
var 書き込む変数の値です。

返り値

書き込みに成功したかどうかを返します。

説明

シーケンスローカル変数に書き込みます。

サウンドハンドルhandleに結びつけられたシーケンスのシーケンスローカル変数に値を書き込みます。 サウンドハンドルが無効の場合は、書き込みに失敗してFALSEを返します。

注意

この関数を呼びだした直後に、 NNS_SndPlayerReadVariable関数で、 同じ変数の値を読み込んでも、書き込んだ値が読みとれる保証はありません。
