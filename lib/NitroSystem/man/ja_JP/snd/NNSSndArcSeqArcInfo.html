NNSSndArcSeqArcInfo

定義

#include <nnsys/snd.h>
typedef struct NNSSndArcSeqArcInfo
{
    u32 fileId;
} NNSSndArcSeqArcInfo;






      
要素

      
      







fileId
サウンドアーカイブ中のファイルIDです。







    



      
説明

      

サウンドアーカイブ中のシーケンスアーカイブ情報構造体です。 


      

NNS_SndArcGetSeqArcInfo関数を使って、
サウンドアーカイブからこの構造体を取得することができます。


    


参照


NNS_SndArcGetSeqArcInfo
