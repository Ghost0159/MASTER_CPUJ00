NNS_SndHandleIsValid

構文

#include <nnsys/snd.h>
BOOL NNS_SndHandleIsValid( const NNSSndHandle* handle );

引数

handle サウンドハンドルのポインタです。

返り値

サウンドハンドルにシーケンスが結びつけられているかどうかを返します。

説明

サウンドハンドルが有効かどうか調べます。サウンドハンドルが有効とは、シーケンスが結びつけられた状態を指します。

参照

なし。

履歴

2007/04/04 引数にconst追加
2004/06/01 サウンドハンドルは、プレイヤーではなくシーケンスと結びつくというように記述修正
