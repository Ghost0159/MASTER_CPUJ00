NNS_SndArcGetSeqArcParam

構文

#include <nnsys/snd.h>
const NNSSndSeqParam* NNS_SndArcGetSeqArcParam(
        int seqArcNo,
        int index );

引数

seqArcNo シーケンスアーカイブ番号です。
index シーケンスアーカイブのインデックス番号です。

返り値

シーケンスパラメータ構造体NNSSndSeqParamの ポインタを返します。 シーケンスアーカイブ番号または、インデックス番号が無効の場合は、NULLを返します。

説明

シーケンスアーカイブ中の各シーケンスに対する、シーケンスパラメータ構造体を取得します。

シーケンスアーカイブ中のシーケンスの個数は、 NNS_SndArcGetSeqArcSeqCount関数で 取得することができます。

注意

あらかじめ、該当シーケンスアーカイブのデータがメモリ上にロードされていなければなりません。 ロードされていない場合は、NULLを返します。

参照

NNSSndSeqParam, NNS_SndArcGetSeqArcSeqCount
