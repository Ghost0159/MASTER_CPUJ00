NNS_SndPlayerSetTrackModSpeed

構文

#include <nnsys/snd.h>
void NNS_SndPlayerSetTrackModSpeed(
        NNSSndHandle* handle,
        u16 trackBitMask,
        int speed );

引数

handle サウンドハンドルのポインタです。
trackBitMask トラックビットマスクです。
speed モジュレーションスピードの値です。値の範囲は、0〜255です。

返り値

なし。

説明

シーケンストラックのモジュレーションスピードを変更します。

サウンドハンドルhandleに、結びつけられたシーケンスのモジュレーションスピードを変更します。 サウンドハンドルが無効の場合は、何もしません。

trackBitMaskで、設定するトラックを指定します。 下位ビットから順に、トラック0、トラック1、トラック2、、、を表し、ビットが立っているトラック全てに対して、 モジュレーションスピードを変更します。 例えば、トラック2とトラック5のモジュレーションスピードを変更したい場合は、 (1 << 2) | (1 << 5) すなわち、0x0024 とします。

speedのデフォルトは16で、約6.4Hzに相当します。 値が２倍になると、周波数も２倍になります。

注意

シーケンスデータ中でも、モジュレーションスピードを設定していた場合は、 後から設定した方が有効になります。

参照

なし。
