NNSSndArcBankInfo

定義

#include <nnsys/snd.h>
typedef struct NNSSndArcBankInfo
{
    u32 fileId;
    u16 waveArcNo[4];
} NNSSndArcBankInfo;






      
要素

      
      







fileId
サウンドアーカイブ中のファイルIDです。




waveArcNo
バンクに関連づけられた波形アーカイブの番号の配列です。







    



      
説明

      

サウンドアーカイブ中のバンク情報構造体です。 


      

NNS_SndArcGetBankInfo関数を使って、
サウンドアーカイブからこの構造体を取得することができます。


      

波形アーカイブ番号の配列waveArcNo中で、
波形アーカイブ番号が指定されていない場合は、
NNS_SND_ARC_INVALID_WAVEARC_NOが入っています。
