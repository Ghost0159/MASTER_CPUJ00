NNS_SndPlayerSetSeqArcNo

構文

#include <nnsys/snd.h>
void NNS_SndPlayerSetSeqArcNo(
        NNSSndHandle* handle,
        int seqArcNo,
        int index );

引数

handle サウンドハンドルのポインタです。
seqArcNo シーケンスアーカイブ番号です。
index シーケンスアーカイブのインデックス番号です。

返り値

なし。

説明

シーケンスアーカイブ番号を設定します。

サウンドハンドルhandleに、結びつけられたシーケンスのシーケンスアーカイブ番号を設定します。 サウンドハンドルが無効の場合は、何もしません。

シーケンスアーカイブ番号を設定すると、後で、設定したシーケンスアーカイブ番号を指定して、シーケンスを止めたりすることができるようになります。 詳しくは、NNS_SndPlayerStopSeqBySeqArcNo関数や、 NNS_SndPlayerCountPlayingSeqBySeqArcNo関数などを参照してください。

注意

この関数は、NNS_SndArcPlayerStartSeqArc*関数内で、 呼びだされています。再度、この関数を呼びだすと、設定した値が上書きされます。

この関数と、NNS_SndPlayerSetSeqNo関数は、 どちらか一方のみ呼びだします。両方呼びだすと、後から設定した方が有効になります。

参照

NNS_SndPlayerStopSeqBySeqArcNo, NNS_SndPlayerCountPlayingSeqBySeqArcNo, NNS_SndArcPlayerStartSeqArc*, NNS_SndPlayerSetSeqNo
