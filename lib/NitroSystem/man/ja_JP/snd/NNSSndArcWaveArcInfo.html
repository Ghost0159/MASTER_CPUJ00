NNSSndArcWaveArcInfo

定義

#include <nnsys/snd.h>
typedef struct NNSSndArcWaveArcInfo
{
    u32 fileId : 24;
    u32 flags  :  8;
} NNSSndArcWaveArcInfo;






      
要素

      
      







fileId
サウンドアーカイブ中のファイルIDです。




flags
オプションフラグです。







    



      
説明

      

サウンドアーカイブ中の波形アーカイブ情報構造体です。 


      

NNS_SndArcGetWaveArcInfo関数を使って、
サウンドアーカイブからこの構造体を取得することができます。


      

オプションフラグflagsは、以下の値の論理和です。


      
      
表 . オプションフラグ







ラベル
値
説明






NNS_SND_ARC_WAVEARC_SINGLE_LOAD
1<<0
この波形アーカイブをロードするとき、
個々の波形データを別々にロードします。
