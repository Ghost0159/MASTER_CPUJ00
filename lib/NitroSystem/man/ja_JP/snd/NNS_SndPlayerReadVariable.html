NNS_SndPlayerReadVariable

構文

#include <nnsys/snd.h>
BOOL NNS_SndPlayerReadVariable(
        NNSSndHandle* handle,
        int varNo,
        s16* var );

引数

handle サウンドハンドルのポインタです。
varNo 変数番号です。値の範囲は、0〜15です。
var 変数の値を格納する先へのポインタです。

返り値

読みとりに成功したかどうかを返します。

説明

シーケンスローカル変数を読みとります。読みとりに成功すると、 varで指定したアドレスへ、変数の値が書き込まれます。

サウンドハンドルhandleに結びつけられたシーケンスのシーケンスローカル変数の値を取得します。 サウンドハンドルが無効の場合は、読みとりに失敗してFALSEを返します。

参照

なし。
