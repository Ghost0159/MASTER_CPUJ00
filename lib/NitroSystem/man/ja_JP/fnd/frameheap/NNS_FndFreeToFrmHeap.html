NNS_FndFreeToFrmHeap

構文

#include <nnsys/fnd.h>
void NNS_FndFreeToFrmHeap(NNSFndHeapHandle heap, int mode);

引数

heap フレームヒープのハンドル
mode メモリブロックの解放方法(下記参照)

返り値

なし。

説明

メモリブロックをまとめて解放します。解放方法には、mode で指定します。mode は下記に示す値をとることが出来ます。

NNS_FND_FRMHEAP_FREE_HEAD ヒープ領域の前方から確保したメモリブロックを一括して解放します。
NNS_FND_FRMHEAP_FREE_TAIL ヒープ領域の後方から確保したメモリブロックを一括して解放します。
NNS_FND_FRMHEAP_FREE_ALL ヒープから確保した全てのメモリブロックを一括して解放します。 NNS_FND_FRMHEAP_FREE_HEAD と NNS_FND_FRMHEAP_FREE_TAIL を同時に指定した場合と同じです。

参照

NNS_FndAllocFromFrmHeap, NNS_FndAllocFromFrmHeapEx
