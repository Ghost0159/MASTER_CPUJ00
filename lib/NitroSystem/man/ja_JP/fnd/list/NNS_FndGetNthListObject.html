NNS_FndGetNthListObject

構文

#include <nnsys/fnd.h>
void* NNS_FndGetNthListObject( NNSFndList* list, u16 index );

引数

list リスト構造体へのポインタ
index オブジェクトのインデックス

返り値

indexで指定されたオブジェクトへのポインタを返します。 オブジェクトが見つからなかった場合には、NULLを返します。

説明

リストの先頭からindex個目に繋がれているオブジェクトへのポインタを返します。 indexに、リストに繋がれているオブジェクトの個数以上の値を指定された場合には、NULLを返します。 この関数では、リストの先頭から順番にオブジェクトをたどるという処理を行いますので、 リストの後方に繋がれているオブジェクトほど、処理に時間がかかります。
