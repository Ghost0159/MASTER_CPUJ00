NNS_G3dGetAnmByName

構文

#include <nnsys/g3d/binres/res_struct_accessor_anm.h>
void*
NNS_G3dGetAnmByName(
    const void*          pRes,
    const NNSG3dResName* pName
);

引数

pRes ロードされたアニメーションファイルへのポインタ
pName リソース名へのポインタ

返り値

pNameで指定されたアニメーションリソースへのポインタを返します。

説明

指定リソース名のアニメーションリソースを取得します。全てのロードされたアニメーションリソースファイルに対して使用できます。みつからなかった場合はNULLを返します。
