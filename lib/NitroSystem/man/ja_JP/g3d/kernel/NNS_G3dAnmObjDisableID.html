NNS_G3dAnmObjDisableID

構文

#include <nnsys/g3d/kernel.h>
void
NNS_G3dAnmObjDisableID(
    NNSG3dAnmObj* pAnmObj,
    int id
);

引数

pAnmObj NNSG3dAnmObj構造体へのポインタ
id ノードID又はマテリアルID

返り値

なし。

説明

指定されたノードＩＤ又はマテリアルＩＤに対するアニメーションを無効にします。デフォルトでは、アニメーションが存在するノードＩＤやマテリアルＩＤに対してはアニメーションは有効になっています。アニメーションの一部だけを再生したい（腕だけをアニメーションさせる等）、といった場合にアニメーションさせないノードIDを指定して、必要ない部分のアニメーションを無効にします。
