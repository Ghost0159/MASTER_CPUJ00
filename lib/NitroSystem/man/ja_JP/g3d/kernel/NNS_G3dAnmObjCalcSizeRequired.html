NNS_G3dAnmObjCalcSizeRequired

構文

#include <nnsys/g3d/kernel.h>
u32
NNS_G3dAnmObjCalcSizeRequired(
    const void*         pAnm,
    const NNSG3dResMdl* pMdl
);

引数

pAnm アニメーションリソースへのポインタ
pMdl モデルリソースへのポインタ

返り値

NNSG3dAnmObj構造体に必要なメモリサイズを返します。

説明

pAnmで指定されたアニメーションリソースと、pMdlで指定されたモデルリソースに対応する NNSG3dAnmObj構造体に必要なメモリサイズを返します。
