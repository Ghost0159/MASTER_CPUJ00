NNS_G3dRenderObjTestFlag

構文

#include <nnsys/g3d/kernel.h>
BOOL
NNS_G3dRenderObjTestFlag(
    const NNSG3dRenderObj*    pRenderObj,
    NNSG3dRenderObjFlag flag
);

引数

pRenderObj RenderObj構造体へのポインタ
flag テストするフラグの論理和

返り値

指定したフラグがすべてセットされている場合はTRUE、そうでなければFALSEを返します。

説明

レンダリングオブジェクトの実行制御フラグがセットされているかどうかテストします。

参照

NNS_G3dRenderObjSetFlag, NNS_G3dRenderObjResetFlag
