NNS_G3dAnmObjEnableID

構文

#include <nnsys/g3d/kernel.h>
void
NNS_G3dAnmObjEnableID(
    NNSG3dAnmObj* pAnmObj,
    int id
);

引数

pAnmObj NNSG3dAnmObj構造体へのポインタ
id ノードID又はマテリアルID

返り値

なし。

説明

指定されたノードＩＤ又はマテリアルＩＤに対するアニメーションを有効にします。デフォルトでは、アニメーションが存在するノードＩＤやマテリアルＩＤに対してはアニメーションは有効になっていますのでこの関数を実行する必要はありません。

NNS_G3dAnmObjDisableID
