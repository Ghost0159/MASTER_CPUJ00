NNS_G3dBindMdlTexEx

構文

#include <nnsys/g3d/kernel.h>
BOOL
NNS_G3dBindMdlTexEx(
    NNSG3dResMdl*        pMdl,
    const NNSG3dResTex*  pTex,
    const NNSG3dResName* pResName
);

引数

pMdl モデルリソースへのポインタ
pTex テクスチャリソースへのポインタ
pResName テクスチャ名へのポインタ

返り値

バインドが成功した場合TRUEを返します。

説明

pMdlで指定されたモデル内で、pResName というテクスチャ名に関連付けられているマテリアルのテクスチャエントリを、 テクスチャブロック内の名前pResNameを持つテクスチャに関連付けます。

参照

NNS_G3dBindMdlTex, NNS_G3dForceBindMdlTex
NNS_G3dReleaseMdlTex, NNS_G3dReleaseMdlTexEx
