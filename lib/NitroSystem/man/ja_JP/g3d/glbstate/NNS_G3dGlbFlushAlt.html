NNS_G3dGlbFlushWVP

構文

#include <nnsys/g3d/glbstate.h>

void NNS_G3dGlbFlushWVP( void );
void NNS_G3dGlbFlushAlt( void );

引数

なし。

返り値

なし。

説明

G3Dライブラリ内部で保持しているグローバル状態の全てをジオメトリエンジンに送信します。 グローバル状態には、射影変換行列、カメラ行列、モデリング行列、ビューポート、ライトの方向と色、デフォルトのマテリアルカラー等があります。

行列に関しては、カレント射影行列に射影変換行列とカメラ行列とモデリング行列が合成された行列が、カレント位置座標行列と方向ベクトル行列に単位行列が設定されます。なお、グローバル状態をジオメトリエンジンに送信した後のマトリックスモードは、POSITION/VECTORモードになります。
