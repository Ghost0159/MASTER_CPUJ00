NNS_G3dGlbGetWV

構文

#include <nnsys/g3d/glbstate.h>
const MtxFx43* NNS_G3dGlbGetSrtCameraMtx( void );
const MtxFx43* NNS_G3dGlbGetWV(void);

引数

なし。

返り値

カメラ行列にBaseSRT行列をかけた行列へのポインタを返します。

説明

G3Dライブラリ内部で保持しているグローバル状態に設定されているカメラ行列に BaseSRT行列をかけた行列へのポインタを返します。 BaseSRT行列をかけた行列が計算されていなかった場合には、 行列を求めてからその行列へのポインタを返します。 このとき同時にNNS_G3dGlbGetInvWV関数で返される逆行列も求めます。

参照

NNS_G3dGlbGetInvWV
