NNS_G3dGlbLookAt

構文

#include <nnsys/g3d/glbstate.h>
void
NNS_G3dGlbLookAt(
    const VecFx32 * camPos,
    const VecFx32 * camUp,
    const VecFx32 * target
);

引数

camPos カメラの位置ベクトルへのポインタ
camUp カメラの上方向のベクトルへのポインタ
target カメラの焦点へのポインタ

返り値

なし。

説明

G3Dライブラリ内部で保持しているグローバル状態に、カメラ行列をセットします。 camUpは正規化されている必要はありません。
