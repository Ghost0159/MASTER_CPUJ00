NNS_G3dGetChildNodeIDList

構文

#include <nnsys/g3d/util.h>
int
NNS_G3dGetChildNodeIDList(
    u8*       idList,
    const u8* sbc,
    u32       nodeID
);

引数

idList 結果を格納する子ノードIDのリストへのポインタ
sbc SBCコード列へのポインタ
nodeID ノードID

返り値

子ノードの数を返します。

説明

nodeIDの子になるノードIDのリストをidListに作成します。 idListは十分な領域が確保されている必要があります。 返り値は子ノードの数です。 みつからなかった場合は0を返し、idListには何も書き込まれません。
