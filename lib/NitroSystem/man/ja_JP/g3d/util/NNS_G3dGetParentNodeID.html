NNS_G3dGetParentNodeID

構文

#include <nnsys/g3d/util.h>
const u8*
NNS_G3dGetParentNodeID(
    int*      parentID,
    const u8* sbc,
    u32       nodeID
);

引数

parentID 親のID
sbc SBCコード列へのポインタ
nodeID ノードID

返り値

nodeIDを設定しているNODEDESC命令へのポインタを返します。

説明

nodeIDの親になるノードのIDを*parentIDに格納します。 返り値は、nodeIDを設定しているNODEDESC命令へのポインタです。 見つからなかった場合はNULLを返し、*parentIDも変更されません。
