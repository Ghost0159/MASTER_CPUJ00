NNS_McsInitPrint

構文

#include <nnsys/mcs.h>
void NNS_McsInitPrint(
         void*   pBuffer,
         u32     size);

引数

pBuffer 文字列出力のための作業用バッファ。NNS_McsPrintf関数ではここで指定したバッファを使用します。
size 文字列出力のための作業用バッファのサイズ(バイト)。

返り値

なし。

説明

mcs用文字列出力関数を使用するための初期化を行います。文字列出力関数NNS_McsPrintfは出力する文字列を作成するために、引数pBufferとsizeで指定したバッファを使用します。関数NNS_McsPrintfが出力できる文字列の最大の長さは引数sizeで指定したサイズ(バイト)となります。

参照

NNS_McsPutString, NNS_McsPrintf
