NNS_G2dEntryNewOamWithAffineIdx

構文

#include <nnsys/g2d/g2d_Oam.h>
BOOL NNS_G2dEntryNewOamWithAffineIdx
                      ( 
                          NNSG2dOamManagerInstance*   pMan, 
                          const GXOamAttr*            pOam, 
                          u16                         affineIdx );

引数

pMan [OUT] OAMマネージャ 実体
pOam [IN] 登録するGXOamAttrのアドレス
affineIdx [IN] 登録するGXOamAttrが使用するアフィンパラメータのインデックス

返り値

登録するのに十分な空き容量がない場合は、まったく登録を行わずFALSEを返します

説明

OBJをOAMマネージャに登録します。その際 OBJの参照するアフィンインデックスを書き換えます
本関数は NNS_G2dEntryOamManagerOamWithAffineIdx() に 改名されました。本関数は、過去のバージョンとの互換性維持のため、別名としてのこされます。

参照

 NNS_G2dEntryOamManagerOam
