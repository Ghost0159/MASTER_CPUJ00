NNS_G2dSetAnimSequenceToMCAnimation

構文

#include <nnsys/g2d/g2d_MultiCellAnimation.h>
void NNS_G2dSetAnimSequenceToMCAnimation( 
                                    NNSG2dMultiCellAnimation*             pMultiCellAnim, 
                                    const NNSG2dMultiCellAnimSequence*    pAnimSeq );

引数

pMultiCellAnim [OUT] マルチセルアニメーション
pAnimSeq [IN] アニメーションシーケンス

返り値

なし

説明

マルチセルアニメーションに再生アニメーションシーケンスを設定します。
アニメーションシーケンスはNNS_G2D_ANIMATIONTYPE_MULTICELLLOCATIONタイプである必要があります。
内部でNNS_G2dGetMCNumNodesRequired()を実行し、セルアニメーション実体がアニメーション再生に十分なノード数を持たない場合はアサートに失敗します。

参照

NNS_G2dGetMCNumNodesRequired
