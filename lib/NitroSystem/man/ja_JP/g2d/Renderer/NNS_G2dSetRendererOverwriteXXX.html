NNS_G2dSetRendererOverwrite*

構文

#include <nnsys/g2d/g2d_Renderer.h>

    NNS_G2D_INLINE void NNS_G2dSetRendererOverwritePriority
    (
        NNSG2dRendererInstance*     pRend,
        u16                         Priority
    );
    
    NNS_G2D_INLINE void NNS_G2dSetRendererOverwritePlttNo
    (
        NNSG2dRendererInstance*     pRend,
        u16                         plttNo
    );
    
    NNS_G2D_INLINE void NNS_G2dSetRendererOverwriteMosaicFlag
    (
        NNSG2dRendererInstance*     pRend,
        BOOL                        overwriteMosaicFlag
    );
    
    NNS_G2D_INLINE void NNS_G2dSetRendererOverwriteOBJMode
    (
        NNSG2dRendererInstance*     pRend,
        GXOamMode                   overwriteObjMode
    );

引数

pRend [OUT] レンダラ 実体
priority [IN] 上書きする描画優先度（0-3）
plttNo [IN] 上書きするパレット番号(0-15)
overwriteMosaicFlag [IN] 上書きするモザイクフラグ
overwriteObjMode [IN] 上書きするオブジェモード（GXOamMode）

返り値

なし

説明

レンダラが描画するOBJに上書きするパラメータの値を設定します。
NNS_G2dSetRendererOverwritePriority()は描画優先度を設定します。
NNS_G2dSetRendererOverwritePlttNo()はパレット番号を設定します。
NNS_G2dSetRendererOverwriteMosaicFlag()はモザイクフラグを設定します。
NNS_G2dSetRendererOverwriteOBJMode()はオブジェモードを設定します。

パラメータ上書き機能を有効にするために、NNS_G2dSetRendererOverwriteEnable()で適切なパラメータの上書き機能を有効に設定する必要があります。

参照

NNS_G2dSetRendererOverwriteEnable/Disable
 NNS_G2dGetRendererOverwrite*
