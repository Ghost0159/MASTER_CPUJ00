NNS_G2dFontSetAlternateChar

構文


#include <nnsys/g2d/g2d_Font.h>

BOOL NNS_G2dFontSetAlternateChar(
	NNSG2dFont* pFont,
	u16 c
);

引数

pFont [IN] フォント へのポインタ
c [IN] 新しい代替文字の文字コード

返り値

代替文字の置き換えに成功すれば TRUE を返します。
フォント内に c に対応するグリフが存在しない場合に失敗します。

説明

フォントの代替文字を変更します。
NNS_G2dFontSetAlternateGlyphIndex() では代替文字をグリフインデックスで指定することができます。

代替文字はフォントに含まれていない文字の代わりとして用いられる文字です。

参照

NNS_G2dFontSetAlternateGlyphIndex, NNS_G2dFontGetAlternateGlyphIndex
