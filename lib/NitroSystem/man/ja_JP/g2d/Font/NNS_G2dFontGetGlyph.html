NNS_G2dFontGetGlyph

構文


#include <nnsys/g2d/g2d_Font.h>

void NNS_G2dFontGetGlyph(
	NNSG2dGlyph* pGlyph,
	const NNSG2dFont* pFont,
	u16 ccode
);

引数

pGlyph [OUT] グリフ情報を格納するバッファへのポインタ
pFont [IN] フォント へのポインタ
ccode [IN] グリフを取得する文字の文字コード

返り値

なし。

説明

グリフの持つ情報をまとめて取得します。

この関数は縦書き/縦持ちフォントに対応していません。
