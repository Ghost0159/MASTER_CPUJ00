NNS_G2dFontGetCharWidths

構文


#include <nnsys/g2d/g2d_Font.h>

const NNSG2dCharWidths* NNS_G2dFontGetCharWidths(
	const NNSG2dFont* pFont,
	u16 c
);

引数

pFont [IN] フォント へのポインタ
c [IN] 文字幅情報を取得する文字の文字コード

返り値

文字幅情報へのポインタを返します。

説明

c に対応するグリフの文字幅情報を取得します。
c に対応するグリフが存在しない場合や、 グリフが固有の文字幅情報を持たない場合はフォントのデフォルト文字幅情報を取得します。

この関数は縦書き/縦持ちフォントに対応していません。
