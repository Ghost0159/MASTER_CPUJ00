NNS_G2dFontGetCharWidth

構文


#include <nnsys/g2d/g2d_Font.h>

const int NNS_G2dFontGetCharWidth(
	const NNSG2dFont* pFont,
	u16 c
);

引数

pFont [IN] フォント へのポインタ
c [IN] 文字幅を取得する文字の文字コード

返り値

ピクセル単位での文字幅を返します。

説明

c に対応するグリフの文字幅を取得します。
c に対応するグリフが存在しない場合や、 グリフが固有の文字幅情報を持たない場合はフォントのデフォルト文字幅を取得します。

縦書きフォントでは文字の高さを取得します。

参照

NNS_G2dFontGetCharWidthFromIndex
