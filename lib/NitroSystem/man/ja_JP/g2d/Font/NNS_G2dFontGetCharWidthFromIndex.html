NNS_G2dFontGetCharWidthFromIndex

構文


#include <nnsys/g2d/g2d_Font.h>

int NNS_G2dFontGetCharWidthFromIndex(
	const NNSG2dFont* pFont,
	u16 idx
);

引数

pFont [IN] フォント へのポインタ
idx [IN] 文字幅を取得するグリフのグリフインデックス

返り値

ピクセル単位での文字幅を返します。

説明

idx が指すグリフの文字幅を取得します。
idx が指すグリフが存在しない場合や、 グリフが固有の文字幅情報を持たない場合はフォントのデフォルト文字幅を返します。

縦書きフォントでは文字の高さを取得します。

参照

NNS_G2dFontGetCharWidth
