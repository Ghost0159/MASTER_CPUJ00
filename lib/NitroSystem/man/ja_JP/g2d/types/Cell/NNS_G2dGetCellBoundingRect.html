NNS_G2dGetCellBoundingRect

構文

#include <nnsys/g2d/fmt/g2d_Cell_data.h>

static inline const NNSG2dCellBoundingRectS16* 
NNS_G2dGetCellBoundingRect( const NNSG2dCellData* pCell );

引数

pCell [IN] セルデータ

返り値

境界矩形(s16フォーマット)情報

説明

セルの境界矩形を取得します。
内部ではセルが実際に境界矩形情報を持つかどうか判定しません。判定はユーザコードで行ってください。
セルが境界矩形情報を持つかどうかは、NNS_G2dCellHasBR()で判定が可能です。
境界矩形情報を持たないセルに対して本関数を実行した場合は不正な情報が返ります。

境界矩形情報を持つセルデータを作成するには、-br オプションを指定してコンバートを行ってください。

NNS_G2dCellHasBR
