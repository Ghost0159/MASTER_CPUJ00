NNS_G2dGetSizeRequiredVramTransferCellDataBank

構文

#include <nnsys/g2d/fmt/g2d_Cell_data.h>

    static inline u32 NNS_G2dGetSizeRequiredVramTransferCellDataBank
    ( 
        const NNSG2dCellDataBank* pCellBank 
    );

引数

pCellBank [IN] セルデータバンク

返り値

VRAM転送に必要なVRAM領域のバイトサイズ

説明

VRAM転送セルバンクが必要とするVRAM領域の大きさ（バイト）を取得します。 VRAM転送情報を持っていない場合はゼロを返します。
VRAM転送セルアニメーションのためのVRAM割り当てなどに使用してください。

本関数はスペルミスを含んだ NNS_G2dGetSizeRequiredVramTransferCellDataBank()から改名された関数です。( Reqired => Required ) 以前の関数名は別名として残されています。

参照
