NNS_G2dGetUserExAnimFrmAttrValue

構文

#include <nnsys/g2d/fmt/g2d_Anim_data.h>

static inline u32
NNS_G2dGetUserExAnimFrmAttrValue
( 
    const NNSG2dUserExAnimFrameAttr* pFrmAttr
);

引数

pFrmAttr [IN] アニメーションフレームユーザ拡張アトリビュート

返り値

u32型のユーザ拡張アトリビュート値

説明

アニメーションフレームユーザ拡張アトリビュート から u32型のユーザ拡張アトリビュート値を取得します。
ユーザ拡張アトリビュートを含むデータを出力するためには、-oua オプションを指定してデータを出力する必要があります。

参照

なし
