NNS_G2dGetAnimSequenceByIdx

構文

#include <nnsys/g2d/load/g2d_NAN_load.h>

const NNSG2dAnimSequenceData* NNS_G2dGetAnimSequenceByIdx( 
    const NNSG2dAnimBankData*   pAnimBank, 
    u16                         idx );

引数

pAnimBank [IN] アニメーション情報バンク
idx [IN] インデックス

返り値

アニメシーケンス情報

説明

アニメーション情報バンクからインデックス指定で、アニメシーケンス情報を取得します。
不正なインデックスが指定された場合はNULLを返します。

参照

なし
