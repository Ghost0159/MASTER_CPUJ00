NNS_G2dGetAnimCtrlType

構文

#include <nnsys/g2d/g2d_Animation.h>
static inline NNSG2dAnimationType NNS_G2dGetAnimCtrlType( 
                                        const NNSG2dAnimController*    pAnimCtrl );

引数

pAnimCtrl [IN] アニメーションコントローラ

返り値

アニメーションコントローラに関連付けられているアニメーションの種類

説明

アニメーションの種類を返します。
ユーザはアニメーションの種類に応じて、アニメーション結果のアクセス方法や、使用の仕方を変化させる必要があります。

本関数はNNS_G2dGetAnimControllerType()から改名された新しい関数名です。互換性維持のため、過去の関数は別名として残されています。
