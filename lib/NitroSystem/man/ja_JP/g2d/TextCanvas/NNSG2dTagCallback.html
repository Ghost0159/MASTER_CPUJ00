NNSG2dTagCallback

構文


#include <nnsys/g2d/g2d_TextCanvas.h>

typedef void (*NNSG2dTagCallback)(
	u16 c,
	NNSG2dTagCallbackInfo* pInfo
);

引数

c [IN] コールバックの発生要因となった文字
pInfo [IN,OUT] 描画情報構造体へのポインタ

返り値

なし。

説明

NNS_G2dTextCanvasDrawTaggedText() からタグの処理のために呼び出されるコールバック関数です。

引数 c にはコールバックの発生要因となった文字（文字コード0x001F以下の文字） が渡されます。 引数 pInfo には NNS_G2dTextCanvasDrawTaggedText() が 内部で描画に用いているパラメータが格納されており、これを書き換えることで、 描画状態を変更する事ができます。

参照

NNS_G2dTextCanvasDrawTaggedText, NNSG2dTagCallbackInfo
