NNS_G2dIsBinFileValid

構文

#include <nnsys/g2d/g2d_load.h> 
static inline BOOL NNS_G2dIsBinFileValid(
                        const NNSG2dBinaryFileHeader*   pBinFile, 
                        u32                             binFileSig, 
                        u16                             version 
                      );

引数

pBinFile [IN] バイナリファイル
binFileSig [IN] バイナリファイル識別子
version [IN] バージョン

返り値

ファイルが目的とするものと合致するならばTRUE

説明

ファイルから読み込んだバイナリファイルが目的とするものと一致するか調査します。

参照

NNS_G2dUnPack*
