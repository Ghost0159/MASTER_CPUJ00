NNS_G2dSetRndCoreImageProxy

構文

#include <nnsys/g2d/g2d_RendererCore.h>

    
    void NNS_G2dSetRndCoreImageProxy
    ( 
        NNSG2dRndCoreInstance*              pRnd,
        const NNSG2dImageProxy*             pImgProxy, 
        const NNSG2dImagePaletteProxy*      pPltProxy
    );

引数

pRnd [OUT] レンダラコア
pImgProxy [IN] 画像プロクシ
pPltProxy [IN] 画像パレットプロクシ

返り値

なし。

説明

レンダラコアモジュールが描画の際に参照する画像プロクシを設定します。
レンダラモジュールとは異なり、Begin-EndRenderingブロック内で本関数を呼び出して、画像プロクシを変更することは不正です。
これは、レンダラコアモジュールがBeginRenderingのタイミングでパラメータ計算を行うためです。

本関数は、Begin-EndRenderingブロックの外で呼び出す必要があります。

参照

NNS_G2dInitRndCore
