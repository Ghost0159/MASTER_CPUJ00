ソケットライブラリサンプルデモ

場所

$NitroWiFi/build/demos/soc

コンパイル方法

$NitroWiFi/build/demos/soc ディレクトリに移り、

% make

と入力します。実行ファイル(.nef, .srl ファイル) が、

$NitroWiFi/build/demos/soc/(サンプル名)/bin/(プラットフォーム)/(ターゲット)/

の下に生成されます。例えば、

$NitroWiFi/build/demos/soc/httpclient/bin/ARM9-TS/Release/

のようになります。

実行方法

$NitroWiFi/build/demos/soc/(サンプル名)/bin/(プラットフォーム)/(ターゲット)/ ディレクトリに移り、以下のうちの一つを行います。

  • ensata で *.nef, *.srl を実行する。
  • IS-NITRO-DEBUGGER で *.nlf ファイルを読みこんで実行する。
  • IS-NITRO-DEBUGGER で *.srl を実行する。

説明

dhcp_monkey SOC ライブラリを使用して DHCP サーバへの接続/切断を繰り返すサンプルです。
ftpclient SOC ライブラリを使用して FTP クライアントとして FTP サーバに接続するサンプルです。
httpclient SOC ライブラリを使用して HTTP クライアントとして HTTP サーバに接続するサンプルです。
httpserver SOC ライブラリを使用して HTTP サーバとして HTTP クライアントと接続をするサンプルです。
telnet SOC ライブラリを使用して telnet に接続するサンプルです。

エミュレータ

すべて動作します。
