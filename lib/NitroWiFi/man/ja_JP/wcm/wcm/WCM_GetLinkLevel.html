WCM_GetLinkLevel

構文


#include <nitroWiFi/wcm.h>

WMLinkLevel  WCM_GetLinkLevel( void );

説明

無線アクセスポイントとの接続リンク強度を取得する同期関数です。

引数

ありません。

返り値

WMLinkLevel 列挙型という形で、無線アクセスポイントとの接続リンク強度 ( 通信電波強度 ) を返します。 WCM ライブラリが初期化前であったり、無線アクセスポイントに接続されていない状態 ( 内部状態が WCM_PHASE_DCF でない場合 ) には WM_LINK_LEVEL_0 を返します。
