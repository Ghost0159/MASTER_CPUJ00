CPSPrivateKey

定義

#include <nitroWiFi/cps.h>
typedef struct {
    int modulus_len;
    u8 *modulus;
    int prime1_len;
    u8 *prime1;
    int prime2_len;
    u8 *prime2;
    int exponent1_len;
    u8 *exponent1;
    int exponent2_len;
    u8 *exponent2;
    int coefficient_len;
    u8 *coefficient;
} CPSPrivateKey;

説明

証明書の私有鍵に関する情報を定義する構造体です。
make_mykeyユーティリティで作成されます。

参照

CPS_SetMyCert, CPSSslConnection
