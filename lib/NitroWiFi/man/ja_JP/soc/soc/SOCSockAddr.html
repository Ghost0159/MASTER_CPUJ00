SOCSockAddr

構文

#include <nitroWiFi/soc.h>
#define SOC_PF_INET   2    // ARPA Internet protocols
typedef struct SOCSockAddr
{
    u8 len;
    u8 family;
    u8 data[6];
} SOCSockAddr;

説明

SOCSockAddr はソケットアドレスを表現するための汎用データ型です。ソケットアドレス構造体を引数として関数に渡すときには、len 要素はsizeof(SOCSockAddr) に設定してください。

注意：DSソケットライブラリではソケットアドレス構造体の大きさを関数の引数にとりません。

参照

SOC_Bind, SOC_Connect, SOC_Accept, SOC_GetSockName, SOCInAddr, SOCSockAddrIn
