はじめに

VoiceChatライブラリを使うことで、ニンテンドーDSのワイヤレス通信（インフラストラクチャーモード）を利用してインターネットに接続し、他のニンテンドーDSのユーザーと音声通話を行うことが出来ます。

VoiceChatライブラリは、下記のライブラリが必要です。

  • NITRO-SDKライブラリ 4.2　+ plus patch(2008/01/18版)
  • NITRO-Systemライブラリ (2007/11/26版) + patch1
  • NITRO-DWCライブラリ 3.0 + plus1 patch(2008/03/05版)
  • NITRO-WiFiライブラリ 2.1 + plus3 patch(2007/09/20版)

また、下記の環境で動作テストしています。

  • CodeWarrior ver.2.0 SP2 + patch2(2008/02/15版)
  • NITRO-SDKライブラリ4.2　+ plus patch(2008/01/18版)
  • NITRO-Systemライブラリ (2007/11/26版) + patch1
  • NITRO-DWCライブラリ3.0 + plus1 patch(2008/03/05版)
  • NITRO-WiFiライブラリ2.1 + plus3 patch(2007/09/20版)

このパッケージには、ソースの含まれないバイナリが含まれますので、ご注意ください。

使用方法

NITROLIBVCT_ROOT 環境変数に、ライブラリへのパスを設定して下さい。

注意事項

VoiceChatライブラリを使用する場合は、NITRO-WiFiを利用したインターネット接続、およびNITRO-DWCを利用したマッチメイクが必須となります。

API の使用

プログラムのソースコードには、以下のヘッダファイルをインクルードして下さい。


	#inculude <vct.h>

アプリケーション・ライブラリを作成する Makefile には、以下の共通ビルドツールをインクルードして下さい。


	include $(NITROLIBVCT_ROOT)/build/buildtools/commondefs
	include $(NITROLIBVCT_ROOT)/build/buildtools/modulerules

このビルドツールをインクルードするとNITRO-WiFi, NITRO-DWCのビルドツールもインクルードされます。
