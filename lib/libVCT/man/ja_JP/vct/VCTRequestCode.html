VCTRequestCode

定義

#include <vct.h>

typedef enum {
	VCT_REQUEST_INVITE = 0,
	VCT_REQUEST_BYE,
	VCT_REQUEST_CANCEL,
	VCT_REQUEST_CONTACT,
	VCT_REQUEST_NOTIFY,
	VCT_REQUEST_END
} VCTRequestCode;

定義

VCT_REQUEST_INVITE 会話要求
VCT_REQUEST_BYE 会話終了要求
VCT_REQUEST_CANCEL 会話要求中止
VCT_REQUEST_CONTACT トランシーバーリクエスト
VCT_REQUEST_NOTIFY トランシーバー通知

説明

SSPのリクエストコードです。VCT_Requestで利用します。

VCT_REQUEST_COTACTとVCT_REQUEST_NOTIFYは内部で利用します。ユーザーはこれらのコードでVCT_Requestを呼び出さないでください。
