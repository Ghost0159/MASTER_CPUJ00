VCT_Request

構文

#include <vct.h>
int VCT_Request( VCTSession *session, VCTRequestCode request );

引数

session リクエストを発行するセッション
request リクエストコード

返り値

0なら成功、失敗の場合は負のエラーコード

説明

SSPのリクエストを発行します。

トランシーバーモードでは、この関数ではなくVCT_Contact、およびVCT_Release関数を利用してください。

参照

VCT_CreateSession, VCT_Response
