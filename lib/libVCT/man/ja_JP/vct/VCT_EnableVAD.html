VCT_EnableVAD

構文

#include <vct.h>
void VCT_EnableVAD( BOOL flag );

引数

flag TRUEでVADをオン、FALSEでVADをオフ

返り値

なし

説明

VAD(Voice Activity Detection)を有効、または無効にします。デフォルトでは、VADは有効になっています。

VADをオンにすると、VCT_Main関数内で音声の判定が行われたうえで、パケットの送信をする・しないを自動的に判定するようになります。
