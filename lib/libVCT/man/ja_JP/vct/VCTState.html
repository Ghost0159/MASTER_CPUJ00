VCTState

定義

#include <vct.h>

typedef enum {
    VCT_STATE_INIT,
    VCT_STATE_OUTGOING,
    VCT_STATE_TALKING,
    VCT_STATE_CONNECTED,
    VCT_STATE_INCOMING,
    VCT_STATE_DISCONNECTING
} VCTState;

定義

VCT_STATE_INIT 初期状態
VCT_STATE_OUTGOING NVITE/CONTACT発行済み
VCT_STATE_TALKING 接続完了、会話中
VCT_STATE_CONNECTED 接続完了、他の人が会話中（トランシーバーモードのみのステート）
VCT_STATE_INCOMING 外部よりリクエストを受信
VCT_STATE_DISCONNECTING 切断処理中

説明

SSPの内部状態を示す列挙型です。
